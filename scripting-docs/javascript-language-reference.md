# <a name="javascript-language-reference"></a>Référence du langage JavaScript

JavaScript est un langage de script qui peut être incorporé dans des pages Web et d’autres applications.  
  
L’implémentation Microsoft de JavaScript est conforme à la spécification de langage ECMAScript 5e édition. Elle fournit également des fonctionnalités supplémentaires qui ne sont pas incluses dans les normes ECMA.

> [!IMPORTANT]
> Microsoft a rejoint l’effort de la communauté afin de faire de [MDN Web docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference) la référence de développement « One-Stop » du Web, en commençant par rediriger les 500 pages de référence de l’API JavaScript de Microsoft de [docs.Microsoft.com](https://docs.microsoft.com) vers leurs homologues MDN et en portant nos informations de référence sur les extensions js héritées à des fins d’historique et de maintenance.
