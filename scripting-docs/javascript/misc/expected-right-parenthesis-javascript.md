# <a name="expected--javascript"></a>')' attendu (JavaScript)
Vous a tenté de placer une expression au sein d’un jeu de parenthèses, mais n’incluez pas la parenthèse fermante. Une expression doit être placée dans un jeu de parenthèses ouvrantes et fermantes. Notez l’utilisation de parenthèses dans l’exemple suivant.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Ajoutez les parenthèses droite à l’expression d’évaluation.
