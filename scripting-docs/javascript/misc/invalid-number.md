# <a name="invalid-number"></a>Nombre non valide
Nombre non valide (ErrbadNumber 1013)  
  
 Vous avez essayé d’utiliser un nombre qui n’est pas valide en JavaScript.  
  
 Pour plus d’informations sur les nombres valides, consultez [objet Number](../../javascript/reference/number-object-javascript.md).
