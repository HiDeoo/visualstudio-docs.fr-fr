# <a name="expected-while"></a>'while' attendu
Vous n’avez pas inclus la condition `while` dans une boucle `do...while`. Une instruction **do** doit avoir un test de `while` correspondant à la fin du bloc de code.  
  
### <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Incluez l’instruction de test `while` après l’accolade fermante.
