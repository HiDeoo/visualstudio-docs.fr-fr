# <a name="iactivescriptauthorprocedure-interface"></a>IActiveScriptAuthorProcedure, interface
En plus des méthodes héritées de `IUnknown`, l’interface `IScriptAuthorProcedure` expose la méthode suivante.  
  
## <a name="methods-in-vtable-order"></a>Méthodes dans l'ordre Vtable  
  
|Méthode|Description|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analyse une procédure de code, ajoute le texte de la procédure de code au moteur de création de scripts et crée un objet `IScriptEntry` qui correspond à la procédure de code.|
