# <a name="iscriptentrygetbody"></a>IScriptEntry::GetBody
Retourne le texte qui correspond au corps d’un `IScriptEntry` bloc de script, d’un bloc de fonction ou d’un scriptlet.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
HRESULT GetBody(  
   BSTR               *pbstr  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `pbstr`  
 à Texte qui se trouve dans le corps de l’un des éléments suivants :  
  
- Bloc de script `IScriptEntry`  
  
- Fonction `IScriptEntry` dans un bloc de fonction  
  
- Un gestionnaire d’événements `IScriptEntry` scriptlet  
  
## <a name="return-value"></a>Valeur de retour  
 Élément `HRESULT`. Les valeurs possibles sont notamment celles figurant dans le tableau suivant.  
  
|valeur|Description|  
|-----------|-----------------|  
|`S_OK`|La méthode a réussi.|
