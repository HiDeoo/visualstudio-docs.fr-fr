# <a name="ijsdebugframegetname-method"></a>IJsDebugFrame::GetName, méthode
Obtient le nom convivial du frame de pile.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>Paramètres  
 `pName`  
 à Nom du frame de pile.
