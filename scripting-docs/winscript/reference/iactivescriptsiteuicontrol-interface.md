# <a name="iactivescriptsiteuicontrol-interface"></a>IActiveScriptSiteUIControl, interface
Fournit une méthode pour obtenir le comportement d’un contrôle d’interface utilisateur.  
  
## <a name="methods"></a>Méthodes  
  
|Méthode|Description|  
|------------|-----------------|  
|[Méthode IActiveScriptSiteUIControl::GetUIBehavior](../../winscript/reference/iactivescriptsiteuicontrol-getuibehavior-method.md)|Obtient un [scriptuichandling, énumération](../../winscript/reference/scriptuichandling-enumeration.md) qui représente la façon dont un contrôle d’interface utilisateur doit-elle être traité.|
