# <a name="idebugdocumentinfo-interface"></a>IDebugDocumentInfo, interface
Fournit des informations sur un document, qui peut être ou ne pas être instancié.  
  
 Outre les méthodes héritées de `IUnknown`, le `IDebugDocumentInfo` interface expose les méthodes suivantes.  
  
## <a name="methods-in-vtable-order"></a>Méthodes dans l'ordre Vtable  
  
|Méthode|Description|  
|------------|-----------------|  
|[IDebugDocumentInfo::GetName](../../winscript/reference/idebugdocumentinfo-getname.md)|Retourne le nom de document spécifié.|  
|[IDebugDocumentInfo::GetDocumentClassId](../../winscript/reference/idebugdocumentinfo-getdocumentclassid.md)|Retourne un CLSID identifiant le type de document.|
