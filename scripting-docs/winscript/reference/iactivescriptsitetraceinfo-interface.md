# <a name="iactivescriptsitetraceinfo-interface"></a>IActiveScriptSiteTraceInfo, interface
Fournit une méthode pour envoyer des informations de trace de script.  
  
## <a name="methods"></a>Méthodes  
  
|Méthode|Description|
|-|-|
|[IActiveScriptSiteTraceInfo::SendScriptTraceInfo Method](../../winscript/reference/iactivescriptsitetraceinfo-sendscripttraceinfo-method.md)|Envoie des informations de suivi qui incluent le type d’événement, le contexte et l’instruction de script.|
