# <a name="profilerexternalobjectaddress-type"></a>PROFILER_EXTERNAL_OBJECT_ADDRESS, type
L’adresse de l’objet externe d’un objet, telles qu’un objet alloué en C++, qui est en dehors du tas JavaScript. Utilisé dans [profiler_heap_object, Structure](../../winscript/reference/profiler-heap-object-structure.md) et [profiler_heap_object_relationship, Structure](../../winscript/reference/profiler-heap-object-relationship-structure.md).  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
typedef void* PROFILER_EXTERNAL_OBJECT_ADDRESS;  
```
