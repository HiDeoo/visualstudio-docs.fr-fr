# <a name="profilerexternalobjectaddress-type"></a>PROFILER_EXTERNAL_OBJECT_ADDRESS, type

Utilisé dans [profiler_heap_object, Structure](../../winscript/reference/profiler-heap-object-structure.md) et [profiler_heap_object_relationship, Structure](../../winscript/reference/profiler-heap-object-relationship-structure.md).
