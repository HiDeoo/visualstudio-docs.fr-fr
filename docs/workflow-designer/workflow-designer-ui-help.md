# <a name="workflow-designer-ui-help"></a>Aide sur l'interface utilisateur de Workflow Designer

Cette section contient les rubriques d’aide contextuelle pour le Concepteur de Workflow. Lorsque vous travaillez dans Visual Studio, vous pouvez accéder à ces rubriques à partir de l’interface utilisateur en appuyant sur **F1** quand une boîte de dialogue du Concepteur de flux de travail est ouverte.

Cette section contient également plusieurs rubriques concernant la navigation et les messages d'erreur.
