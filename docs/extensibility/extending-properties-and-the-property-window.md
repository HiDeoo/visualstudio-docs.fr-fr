# <a name="extend-properties-and-the-property-window"></a>Étendre les propriétés et la fenêtre de la propriété
La fenêtre **Propriétés** est utilisée pour visualiser et modifier les propriétés et les événements de conception-temps d’objets sélectionnés.

 Lorsque vous sélectionnez [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]un objet dans , les propriétés publiques de cet objet apparaissent dans la fenêtre **Propriétés.** Pour plus d’informations, voir [aperçu de la fenêtre Propriétés](../extensibility/internals/properties-window-overview.md).
