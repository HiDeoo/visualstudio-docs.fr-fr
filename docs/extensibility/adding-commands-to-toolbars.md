# <a name="add-commands-to-toolbars"></a>Ajouter des commandes aux barres d’outils
Une barre d’outils est une bande horizontale ou verticale qui contient les boutons liés aux commandes. Vous pouvez ajouter vos propres commandes aux barres d’outils dans de nombreuses fenêtres différents dans Visual Studio : la fenêtre principale de Visual Studio, l’Explorateur de solutions et vos propres fenêtres Outil personnalisées.  
  
 Cette section présente un nombre de choses à que faire avec les commandes des barres d’outils.
