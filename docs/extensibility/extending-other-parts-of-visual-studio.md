# <a name="extend-other-parts-of-visual-studio"></a>Étendre d’autres parties de Visual Studio

Il ya beaucoup plus de parties de l’interface utilisateur Visual Studio que vous pouvez étendre. Ici, nous vous montrons quelques-uns.

## <a name="create-a-vspackage"></a>Créer un VSPackage

Les éléments de base de Visual Studio Extensibility sont VSPackages.  Apprenez à ajouter un VSPackage : [Créez une extension avec un VSPackage](../extensibility/creating-an-extension-with-a-vspackage.md)

## <a name="extend-the-toolbox"></a>Étendre la boîte à outils

Découvrez comment ajouter de nouveaux contrôles et d’autres éléments à la boîte à outils et comment utiliser la fonctionnalité Toolbox :

- [Créez un contrôle de boîte à outils WPF](../extensibility/creating-a-wpf-toolbox-control.md)

- [Créer un contrôle windows Forme boîte à outils](../extensibility/creating-a-windows-forms-toolbox-control.md)

## <a name="extend-the-status-bar"></a>Étendre la barre d’état

Apprenez à lire et à écrire à la barre de statut et à la barre de progression, et comment fournir des animations et autres interfaces utilisateur : [Étendre la barre d’état.](../extensibility/extending-the-status-bar.md)

::: moniker range="vs-2017"

## <a name="create-custom-start-pages"></a>Créez des pages de démarrage personnalisées

Apprenez à faire votre propre page de départ, soit à partir de zéro ou à partir d’un échantillon de page de démarrage téléchargeable: [Créer une page de démarrage personnalisée](../extensibility/creating-a-custom-start-page.md).

::: moniker-end

## <a name="write-to-the-activity-log"></a>Écrivez au journal d’activité

Apprenez à écrire au journal d’activité : [Comment : Utilisez le journal d’activité](../extensibility/how-to-use-the-activity-log.md).
