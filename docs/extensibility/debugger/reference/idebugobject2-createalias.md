# <a name="idebugobject2createalias"></a>IDebugObject2::CreateAlias
Crée un ID ou un alias unique pour cet objet ou renvoie un alias existant.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT CreateAlias(
   IDebugAlias** ppAlias
);
```

```csharp
int CreateAlias(
   out IDebugAlias ppAlias
);
```

## <a name="parameters"></a>Paramètres
`ppAlias`\
[out] Le nouveau (ou existant) alias.

## <a name="return-value"></a>Valeur de retour
 En cas de succès, les retours S_OK; autrement, renvoie un code d’erreur.

## <a name="remarks"></a>Notes
 Un alias est une étiquette qui représente un objet particulier pendant que l’objet est en mémoire.
