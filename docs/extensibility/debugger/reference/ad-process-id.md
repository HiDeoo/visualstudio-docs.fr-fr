# <a name="ad_process_id"></a>AD_PROCESS_ID
Spécifie l’ID du processus, qui peut être soit une pièce d’identité système ou un GUID.

## <a name="syntax"></a>Syntaxe

```cpp
typedef struct _AD_PROCESS_ID {
    AD_PROCESS_ID_TYPE ProcessIdType;
    union {
        DWORD dwProcessId; 
        GUID  guidProcessId; 
        DWORD dwUnused; 
    } ProcessId;
} AD_PROCESS_ID;
```

```csharp
public struct AD_PROCESS_ID {
    AD_PROCESS_ID_TYPE ProcessIdType;
    DWORD              dwProcessId; 
    GUID               guidProcessId; 
    DWORD              dwUnused; 
};
```

## <a name="members"></a>Membres
`ProcessIdType`\
Une valeur de [l’énumération AD_PROCESS_ID_TYPE](../../../extensibility/debugger/reference/ad-process-id-type.md) précisant comment `ProcessId` interpréter le syndicat (ou, pour le code géré, quel membre de la structure d’accès).

`dwProcessId`\
L’ID de processus comme une valeur du système.

`guidProcessId`\
L’ID du processus en tant que GUID.

dwUnused Padding.
