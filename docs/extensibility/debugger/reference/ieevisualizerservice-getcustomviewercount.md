# <a name="ieevisualizerservicegetcustomviewercount"></a>IEEVisualizerService::GetCustomViewerCount
Cette méthode obtient le nombre de visualisateurs de type disponibles à partir de ce service.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetCustomViewerCount(
   ULONG* pcelt
);
```

```csharp
int GetCustomViewerCount(
   out uint pcelt
);
```

## <a name="parameters"></a>Paramètres
`pcelt`\
[out] Retourne le nombre d’visualisateurs de type disponibles.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.

## <a name="remarks"></a>Notes
- [GetCustomViewerCount](../../../extensibility/debugger/reference/idebugproperty3-getcustomviewercount.md) transmet la demande à cette méthode dans son support pour les visualisateurs de type.
