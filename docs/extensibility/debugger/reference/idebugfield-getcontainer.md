# <a name="idebugfieldgetcontainer"></a>IDebugField::GetContainer
Cette méthode obtient le conteneur d’un champ.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetContainer( 
   IDebugContainerField** ppContainerField
);
```

```csharp
int GetContainer(
   out IDebugContainerField ppContainerField
);
```

## <a name="parameters"></a>Paramètres
`ppContainerField`\
[out] Retourne le conteneur tel que représenté par l’interface [IDebugContainerField.](../../../extensibility/debugger/reference/idebugcontainerfield.md)

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.

## <a name="remarks"></a>Notes
 Si ce champ n’a pas `ppContainerField` de conteneur, le retour sera une valeur nulle.
