# <a name="idebugcoreserver3getconnectionprotocol"></a>IDebugCoreServer3::GetConnectionProtocol
Retourne une valeur indiquant le protocole utilisé pour communiquer entre le serveur et le paquet de déboise.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetConnectionProtocol(
   CONNECTION_PROTOCOL* pProtocol
);
```

```csharp
int GetConnectionProtocol(
   CONNECTION_PROTOCOL[] pProtocol
);
```

## <a name="parameters"></a>Paramètres
`pProtocol`\
[out] Retourne l’une des valeurs de [l’énumération CONNECTION_PROTOCOL.](../../../extensibility/debugger/reference/connection-protocol.md)

## <a name="return-value"></a>Valeur de retour
 En cas `S_OK`de succès, les retours; autrement, renvoie le code d’erreur.
