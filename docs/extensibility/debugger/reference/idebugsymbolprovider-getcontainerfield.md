# <a name="idebugsymbolprovidergetcontainerfield"></a>IDebugSymbolProvider::GetContainerField
Cette méthode obtient le champ qui contient l’adresse de débogé.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetContainerField( 
   IDebugAddress*         pAddress,
   IDebugContainerField** ppContainerField
);
```

```csharp
int GetContainerField(
   IDebugAddress            pAddress,
   out IDebugContainerField ppContainerField
);
```

## <a name="parameters"></a>Paramètres
`pAddress`\
[dans] L’adresse représentée par une interface [IDebugAddress.](../../../extensibility/debugger/reference/idebugaddress.md)

`ppContainerField`\
[out] Retourne un champ de conteneurs représenté par une interface [IDebugContainerField.](../../../extensibility/debugger/reference/idebugcontainerfield.md)

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.
