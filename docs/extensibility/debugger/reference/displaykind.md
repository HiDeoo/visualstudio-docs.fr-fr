# <a name="displaykind"></a>DisplayKind
Énumère les valeurs valides qui représentent les types d’informations à prendre à partir d’un objet [IDebugField](../../../extensibility/debugger/reference/idebugfield.md) et l’affichage à l’utilisateur.

## <a name="syntax"></a>Syntaxe

```cpp
enum enum_DisplayKind
{
    DisplayKind_Value = 0x1,
    DisplayKind_Name = 0x2,
    DisplayKind_Type = 0x3,
};
typedef DWORD DisplayKind;
```

```csharp
public enum enum_DisplayKind
{
    DisplayKind_Value = 0x1,
    DisplayKind_Name = 0x2,
    DisplayKind_Type = 0x3,
};
```

## <a name="fields"></a>Champs
`DisplayKind_Value`\
Valeur du champ.

`DisplayKind_Name`\
Nom du champ.

`DisplayKind_Type`\
Type de champ.
