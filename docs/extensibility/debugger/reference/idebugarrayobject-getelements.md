# <a name="idebugarrayobjectgetelements"></a>IDebugArrayObject::GetElements
Obtient un enumérateur de tous les éléments du tableau.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetElements( 
   IEnumDebugObjects** ppEnum
);
```

```csharp
int GetElements(
   out IEnumDebugObjects ppEnum
);
```

## <a name="parameters"></a>Paramètres
`ppEnum`\
[out] Retourne un objet [IEnumDebugObjects](../../../extensibility/debugger/reference/ienumdebugobjects.md) qui permet d’énumérer tous les éléments.

## <a name="return-value"></a>Valeur de retour
 En cas de succès, les retours S_OK; autrement, renvoie un code d’erreur.

## <a name="remarks"></a>Notes
 Comme alternative, utilisez les méthodes [GetCount](../../../extensibility/debugger/reference/idebugarrayobject-getcount.md) et [GetElement](../../../extensibility/debugger/reference/idebugarrayobject-getelement.md) pour itérer à travers les éléments.
