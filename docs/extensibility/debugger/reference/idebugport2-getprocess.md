# <a name="idebugport2getprocess"></a>IDebugPort2::GetProcess
Obtient le processus spécifié en cours d’exécution sur un port.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetProcess( 
   AD_PROCESS_ID    ProcessId,
   IDebugProcess2** ppProcess
);
```

```csharp
int GetProcess( 
   AD_PROCESS_ID      ProcessId,
   out IDebugProcess2 ppProcess
);
```

## <a name="parameters"></a>Paramètres
`ProcessId`\
[dans] Une structure [AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md) qui spécifie l’identifiant du processus.

`ppProcess`\
[out] Renvoie un objet [IDebugProcess2](../../../extensibility/debugger/reference/idebugprocess2.md) représentant le processus.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.
