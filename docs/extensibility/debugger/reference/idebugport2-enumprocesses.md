# <a name="idebugport2enumprocesses"></a>IDebugPort2::EnumProcesses
Retourne une liste de tous les processus en cours d’exécution sur un port.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT EnumProcesses( 
   IEnumDebugProcesses2** ppEnum
);
```

```csharp
int EnumProcesses( 
   out IEnumDebugProcesses2 ppEnum
);
```

## <a name="parameters"></a>Paramètres
`ppEnum`\
[out] Retourne un objet [IEnumDebugProcesses2](../../../extensibility/debugger/reference/ienumdebugprocesses2.md) qui contient une liste de tous les processus en cours d’exécution sur un port.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.
