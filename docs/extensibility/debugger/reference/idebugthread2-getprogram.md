# <a name="idebugthread2getprogram"></a>IDebugThread2::GetProgram
Obtient le programme dans lequel un thread est en cours d’exécution.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetProgram ( 
   IDebugProgram2** ppProgram
);
```

```csharp
int GetProgram ( 
   out IDebugProgram2 ppProgram
);
```

## <a name="parameters"></a>Paramètres
`ppProgram`\
[out] Retourne un objet [IDebugProgram2](../../../extensibility/debugger/reference/idebugprogram2.md) qui représente le programme dans lequel ce fil est en cours d’exécution.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.
