# <a name="idebugprocess2getphysicalprocessid"></a>IDebugProcess2::GetPhysicalProcessId
Obtient l’identifiant de processus de système.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetPhysicalProcessId(
   AD_PROCESS_ID* pdwProcessId
);
```

```csharp
int GetPhysicalProcessId(
   AD_PROCESS_ID[] pdwProcessId
);
```

## <a name="parameters"></a>Paramètres
`pdwProcessId`\
[out] Une [structure AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md) qui est remplie avec les informations d’identification de processus système.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.
