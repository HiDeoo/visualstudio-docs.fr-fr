# <a name="ieevisualizerdataprovidergetobjectforvisualizer"></a>IEEVisualizerDataProvider::GetObjectForVisualizer
Cette méthode obtient l’objet que représente ce visualisateur.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetObjectForVisualizer(
   IDebugObject** ppObject
);
```

```csharp
int GetObjectForVisualizer(
   out IDebugObject ppObject
);
```

## <a name="parameters"></a>Paramètres
`ppObject`\
[out] L’objet représenté par ce visualisateur

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.

## <a name="remarks"></a>Notes
 `GetObjectForVisualizer`est autorisé à retourner une version mise en cache de l’objet. Si l’appelant veut s’assurer que l’objet est à jour, alors il appellera [GetNewObjectForVisualizer](../../../extensibility/debugger/reference/ieevisualizerdataprovider-getnewobjectforvisualizer.md).
