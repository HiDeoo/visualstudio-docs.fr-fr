# <a name="idebugprogram2enumcodecontexts"></a>IDebugProgram2::EnumCodeContexts
Récupère une liste des contextes de code pour une position donnée dans un fichier source.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT EnumCodeContexts( 
   IDebugDocumentPosition2*  pDocPos,
   IEnumDebugCodeContexts2** ppEnum
);
```

```csharp
int EnumCodeContexts( 
   IDebugDocumentPosition2     pDocPos,
   out IEnumDebugCodeContexts2 ppEnum
);
```

## <a name="parameters"></a>Paramètres
`pDocPos`\
[dans] Un objet [IDebugDocumentPosition2](../../../extensibility/debugger/reference/idebugdocumentposition2.md) représentant une position abstraite dans un fichier source connu de l’IIDE.

`ppEnum`[out] Retourne un objet [IEnumDebugCodeContexts2](../../../extensibility/debugger/reference/ienumdebugcodecontexts2.md) qui contient une liste des contextes de code.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.

## <a name="remarks"></a>Notes
 Cette méthode permet au gestionnaire de débogé de session (SDM) ou à l’IDE de cartographier une position de fichier source en position de code. Plus d’un contexte de code est retourné si la source génère plusieurs blocs de code (par exemple, les modèles CMD).
