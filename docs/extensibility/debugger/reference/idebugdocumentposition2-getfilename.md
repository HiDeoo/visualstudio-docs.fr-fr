# <a name="idebugdocumentposition2getfilename"></a>IDebugDocumentPosition2::GetFileName
Obtient le nom de fichier du fichier source qui contient la position du document.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetFileName( 
   BSTR* pbstrFileName
);
```

```csharp
int GetFileName( 
   out string pbstrFileName
);
```

## <a name="parameters"></a>Paramètres
`pbstrFileName`\
[out] Renvoie le nom de fichier du fichier source.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.

## <a name="remarks"></a>Notes
 Un fichier source peut ne pas toujours avoir un nom de fichier (le fichier source peut ne pas exister sur le disque, par exemple).
