# <a name="idebugcoreserver2getportsupplier"></a>IDebugCoreServer2::GetPortSupplier
Récupère un fournisseur de port spécifique.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetPortSupplier( 
   REFGUID               guidPortSupplier,
   IDebugPortSupplier2** ppPortSupplier
);
```

```csharp
int GetPortSupplier( 
   ref Guid                guidPortSupplier,
   out IDebugPortSupplier2 ppPortSupplier
);
```

## <a name="parameters"></a>Paramètres
`guidPortSupplier`\
[dans] GUID du fournisseur de port à récupérer.

`ppPortSupplier`\
[out] Renvoie un objet [IDebugPortSupplier2](../../../extensibility/debugger/reference/idebugportsupplier2.md) représentant le fournisseur portuaire souhaité.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.
