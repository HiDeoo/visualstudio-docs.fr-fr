# <a name="idebugprocess2getserver"></a>IDebugProcess2::GetServer
Obtient le serveur que ce processus est en cours d’exécution sur.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT GetServer( 
   IDebugCoreServer2** ppServer
);
```

```csharp
int GetServer( 
   out IDebugCoreServer2 ppServer
);
```

## <a name="parameters"></a>Paramètres
`ppServer`\
[out] Retourne un objet [IDebugCoreServer2](../../../extensibility/debugger/reference/idebugcoreserver2.md) qui représente le serveur sur lequel ce processus est en cours d’exécution.

## <a name="return-value"></a>Valeur de retour
 En cas de réussite, retourne `S_OK` , sinon, retourne un code d'erreur.

## <a name="remarks"></a>Notes
 Plus d’un serveur peut fonctionner sur une seule machine.
