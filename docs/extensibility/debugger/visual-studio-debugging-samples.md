# <a name="visual-studio-debugging-samples"></a>Échantillons de débogage Visual Studio
Deux échantillons montrent comment étendre le debugger Visual Studio en utilisant [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)].

- L’échantillon d’évaluateur d’expression de Debugger, pour les langues gérées, est inclus dans le [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)]. Dans [!INCLUDE[vsipsdk](../../extensibility/includes/vsipsdk_md.md)] la fenêtre, cliquez sur l’onglet **Échantillons,** cliquez sur **Tous** ou **C** dans la liste des langues, puis cliquez sur **Debugger**.

- Le Visual Studio Debug Engine Sample, pour les langues autochtones, est disponible sur MSDN Code Gallery. Pour plus d’informations, consultez [l’échantillon de moteur de débogé de Visual Studio](https://code.msdn.microsoft.com/Visual-Studio-Debug-Engine-c2e21c0e) sur le site Web de Microsoft.
