# <a name="asyncvoidmethodbuilderm_objectidfordebugger-field"></a>AsyncVoidMethodBuilder.m_objectIdForDebugger champ
Représente l’objet paresseusement initialisé utilisé par le débbugeur pour identifier de façon unique ce constructeur.

 **Espace nom:**<xref:System.Runtime.CompilerServices?displayProperty=fullName>

 **Assemblée:** mscorlib (en mscorlib.dll)

 Parce que vous ne pouvez pas accéder à ce membre interne à partir du cadre .NET, la syntaxe suivante est fournie dans common Intermediate Language (CIL).

```csharp
.field private object m_objectIdForDebugger
```
