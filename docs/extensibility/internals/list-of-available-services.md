# <a name="list-of-available-services"></a>Liste des services disponibles

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]et le Visual Studio SDK prennent en charge les services suivants. Certains forfaits offrent leurs propres services qui ne sont pas répertoriés ici, par exemple, les services linguistiques n’ont pas un seul service GUID. Vous devez utiliser le nom de la langue pour trouver le GUID du service linguistique dans le registre.

Utilisez les GUID de service répertoriés ici ou obtenus auprès d’une autre source (par exemple, les services linguistiques) pour obtenir l’interface principale ou les interfaces affichées avec chaque service.
