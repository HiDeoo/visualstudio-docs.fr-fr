# <a name="get-started-with-source-control-vspackages"></a>Prise en main le contrôle de code source VSPackages
Un VSPackage de contrôle de code source est une alternative plus avancée pour le plug-in de contrôle de code source. Pour plus d’informations sur les plug-ins de contrôle de code source, consultez [créer un plug-in de contrôle de code source] (.. /.. / extensibility/internals/creating-a-source-control-plug-in.md. Contrôle total sur le modèle de contrôle source, fonctionnalités et l’interface utilisateur (IU) permet à un VSPackage de contrôle de code source, et s’intègre dans le [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] environnement en tant qu’un VSPackage.
