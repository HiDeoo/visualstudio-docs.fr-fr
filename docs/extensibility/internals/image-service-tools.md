# <a name="image-service-tools"></a>Outils de service d’image
Le VS SDK comprend plusieurs outils pour aider les constructeurs d’extension à utiliser le service d’image Visual Studio.

- [L’outil Manifest from Resources](../../extensibility/internals/manifest-from-resources.md) prend une liste de ressources d’image (.fichiers ou .xaml) et génère un fichier manifeste d’image.

- [L’outil Manifeste to Code](../../extensibility/internals/manifest-to-code.md) prend un fichier manifeste d’image et génère un fichier d’emballage à utiliser dans les fichiers C, C, VB ou .vsct.

- Le [Visual de bibliothèque d’images](../../extensibility/internals/image-library-viewer.md) peut charger, modifier et rechercher des manifestes d’image.
