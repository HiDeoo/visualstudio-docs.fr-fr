# <a name="color-theme-tools"></a>Outils de thème de couleur
Le SDK Visual Studio fournit plusieurs outils pour vous aider à créer et gérer des thèmes de couleurs.  
  
-   Le [de couleurs VSIX](../../extensibility/internals/vsix-color-editor.md) outil peut créer et modifier des couleurs personnalisées pour Visual Studio.  
  
-   Le [compilateur de couleur VSIX](../../extensibility/internals/vsix-color-compiler.md) outil convertit le thème de Visual Studio existant *.xml* de fichiers dans *.pkgdef* fichiers pour votre extension.
