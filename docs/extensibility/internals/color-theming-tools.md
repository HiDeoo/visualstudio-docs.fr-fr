# <a name="color-theme-tools"></a>Outils de thème de couleur
Le Visual Studio SDK fournit plusieurs outils pour vous aider à créer et à gérer des thèmes de couleur.

- [L’outil VSIX Color Editor](../../extensibility/internals/vsix-color-editor.md) peut créer et modifier des couleurs personnalisées pour Visual Studio.

- [L’outil VSIX Color Compiler](../../extensibility/internals/vsix-color-compiler.md) convertit les fichiers Visual Studio existants *.xml* thème en fichiers *.pkgdef* pour votre extension.
