# <a name="source-control"></a>Contrôle de la source
Cette section traite des options de mise en [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]œuvre du contrôle des sources comme caractéristique intégrée de , soit par le biais d’un plug-in de contrôle source ou d’un contrôle source VSPackage.

## <a name="in-this-section"></a>Dans cette section
- [Bases](../../extensibility/internals/source-control-integration-essentials.md)

 Présente les informations importantes que vous devez commencer à travailler avec le contrôle des sources.

- [Vue d’ensemble](../../extensibility/internals/source-control-integration-overview.md)

 Un aperçu des deux options disponibles pour la mise en œuvre du contrôle des sources.

- [Création d’un plug-in de contrôle de code source](../../extensibility/internals/creating-a-source-control-plug-in.md)

 Discute de la façon de créer un plug-in [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] de contrôle source qui fournit la fonctionnalité de contrôle des sources grâce à l’interface utilisateur de contrôle source (interface utilisateur).

- [Création d’un VSPackage de contrôle de code source](../../extensibility/internals/creating-a-source-control-vspackage.md)

 Discute de la façon de créer un contrôle source VSPackage qui non [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] seulement fournit la fonctionnalité de contrôle des sources, mais peut être utilisé pour personnaliser l’interface utilisateur de contrôle source.
