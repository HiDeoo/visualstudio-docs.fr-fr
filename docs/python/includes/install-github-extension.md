Pour utiliser des dépôts GitHub depuis VS, vous devez installer l’extension GitHub pour Visual Studio. Pour ce faire, exécutez le programme d’installation de Visual Studio, sélectionnez **modifier**, puis sélectionnez l’onglet **composants individuels** . Faites défiler jusqu’à la section **outils de code** , sélectionnez **extension GitHub pour Visual Studio**, puis sélectionnez **modifier**.
