Pour utiliser des dépôts GitHub depuis VS, vous devez installer l’extension GitHub pour Visual Studio. Pour ce faire, exécutez l’installateur Visual Studio, **sélectionnez Modifier**et sélectionnez **l’onglet Composants individuels.** Faites défiler vers le bas vers le bas à la section **des outils de code,** sélectionnez **l’extension GitHub pour Visual Studio**, et **sélectionnez Modifier**.
