# <a name="getting-started-with-ptvs-setting-up-visual-studio"></a>Prise en main de PTVS : configuration de Visual Studio

[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Si vous avez Visual Studio, l'installation de PTVS et des bibliothèques associées est une opération rapide. Si vous ne disposez pas de Visual Studio, vous pouvez obtenir une copie gratuite d'une version de qualité professionnelle.

Vous pouvez obtenir des instructions en regardant cette très courte [vidéo youtube](https://www.youtube.com/watch?v=_okUV47eM5c&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=1).

Les étapes générales sont les suivantes : installer Visual Studio, installer PTVS, installer Python et les bibliothèques de données (Anaconda, Canopy), puis vérifier l’installation.

La première chose dont vous avez besoin est Visual Studio. Si vous êtes développeur open source ou indépendant, vous pouvez utiliser Visual Studio [Community Edition](https://www.visualstudio.com/products/visual-studio-community-vs). Cette version gratuite et opérationnelle s’adresse aux professionnels. L’utilisation de la version Community Edition dans une organisation est limitée à la formation en salle de classe, aux travaux de recherche académique et aux contributions à des projets open source. Si vous êtes étudiant ou propriétaire d'une jeune entreprise, examinez les programmes DreamSpark et BizSpark pour voir si vous pouvez bénéficier d'un accès gratuit, ou demandez à votre employeur si vous disposez d'un abonnement MSDN.

Une fois que vous avez Visual Studio, vous devez [installer PTVS](https://archive.codeplex.com/?p=pytools). Autonome et gratuite, cette extension est entièrement prise en charge par Microsoft et développée de manière ouverte grâce aux contributions de la communauté.

Vous devez maintenant [installer Python](https://www.python.org/download/). Python est entièrement géré par la communauté, et sa page d’accueil se trouve sur python.org. Continuum Analytics propose une offre groupée gratuite appelée Anaconda qui intègre Python et plusieurs bibliothèques utiles (en particulier dans le domaine des sciences et du traitement des données). Enthought produit une offre groupée similaire appelée Canopy. Vous n’avez besoin d’installer qu’un seul de ces produits. Si vous ne savez pas laquelle choisir, commencez par [Anaconda](https://www.continuum.io/downloads). Celle-ci contient la version la plus récente de Python et la plupart des packages difficiles à installer.

Démarrez Visual Studio et assurez-vous que tout fonctionne correctement. Dans le menu Affichage, sélectionnez Autres fenêtres. La liste contient un élément nommé Environnements Python. Cette fenêtre affiche toutes les installations de Python détectées par PTVS et tous les packages installés. La fenêtre contrôle également l'actualisation des bases de données pour afficher la saisie semi-automatique lorsque vous modifiez du code. Ce processus d’actualisation prend du temps, mais une fois effectué, PTVS peut afficher davantage d’informations utiles sur les packages.

Si vous voulez utiliser IPython avec PTVS, suivez ces [instructions](https://archive.codeplex.com/?p=pytools).

Vous pouvez obtenir ces instructions en regardant cette courte [vidéo YouTube](https://www.youtube.com/watch?v=_okUV47eM5c&list=PLReL099Y5nRdLgGAdrb_YeTdEnd23s6Ff&index=1).
