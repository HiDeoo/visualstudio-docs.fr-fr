# <a name="how-to-open-the-data-sources-window"></a>Comment : ouvrir la fenêtre Sources de données
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Le **des Sources de données** fenêtre affiche les sources de données dans votre projet. Vous utilisez le **des Sources de données** fenêtre pour créer une interface utilisateur de contrôles liés aux données en faisant glisser des éléments de la fenêtre sur les formulaires dans votre projet. Consultez [lier les formulaires Windows contrôle aux données dans Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 Vous pouvez remplir le **des Sources de données** fenêtre en exécutant la **Configuration de Source de données** Assistant. Pour plus d’informations, consultez [Assistant de Configuration de Source de données](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f).  
  
### <a name="to-open-the-data-sources-window"></a>Pour ouvrir la fenêtre Sources de données  
  
-   Dans la barre de menus, choisissez **vue**, **Windows autres**, **des Sources de données** (ou appuyez sur les touches Maj + Alt + D).  
  
    > [!IMPORTANT]
    >  Vous pouvez ajouter une source de données à un projet uniquement s'il prend en charge la création et l'utilisation des sources de données. Par exemple, vous ne pouvez pas ouvrir le **des Sources de données** fenêtre dans un projet pour une application Windows Store.
