# <a name="designing-datatables"></a>Conception de DataTables
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../includes/vsprvs-md.md)] Fournit des outils de conception pour créer et modifier des tables de données (**DataTable**). Pour une vue d’ensemble de ce à quoi un **DataTable** , consultez [DataTables](http://msdn.microsoft.com/library/52ff0e32-3e5a-41de-9a3b-7b04ea52b83e).  
  
 Les rubriques suivantes expliquent comment créer des tables de données et les ajouter à l’utilisation de datasets typés le **Concepteur de Dataset**et créer et modifier les colonnes de données (**DataColumn**) qui les définissent.  
  
> [!NOTE]
>  Les boîtes de dialogue et les commandes de menu qui s'affichent peuvent être différentes de celles qui sont décrites dans l'aide, en fonction de vos paramètres actifs ou de l'édition utilisée. Pour modifier vos paramètres, choisissez **Importation et exportation de paramètres** dans le menu **Outils** . Pour plus d’informations, consultez [Personnalisation des paramètres de développement dans Visual Studio](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## <a name="reference"></a>Référence  
 <xref:System.Data.DataSet>  
 Représente un cache de données en mémoire.  
  
 <xref:System.Data.DataTable>  
 Représente une table de données en mémoire.  
  
 <xref:System.Data.DataColumn>  
 Représente le schéma d’une colonne dans un <xref:System.Data.DataTable>.  
  
 <xref:System.Data.DataRelation>  
 Représente une relation parent/enfant entre deux <xref:System.Data.DataTable> objets.
