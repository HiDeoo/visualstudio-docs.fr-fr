# <a name="fetching-data-into-your-application"></a>Récupération de données dans votre application
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Chargement des données dans votre application décrit le processus de récupération des données à partir d’une source de données, tel qu’une base de données ou un service et de remplissage des objets dans le modèle de données d’une application. Les rubriques suivantes décrivent comment extraire des données et charger des données dans les modèles de données disponibles.

|[Utilisation d’un modèle conceptuel](../data-tools/working-with-a-conceptual-model-wcf-data-services.md)|Décrit comment interroger des données qui sont définies par un modèle conceptuel à l’aide de LINQ et les procédures stockées.|
