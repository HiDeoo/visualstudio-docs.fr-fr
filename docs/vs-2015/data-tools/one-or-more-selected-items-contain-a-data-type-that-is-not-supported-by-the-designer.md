# <a name="one-or-more-selected-items-contain-a-data-type-that-is-not-supported-by-the-designer"></a>Un ou plusieurs éléments sélectionnés contient un type de données non pris en charge par le concepteur
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Un ou plusieurs des éléments déplacés à partir de **Explorateur de serveurs**/**Database Explorer** sur le [!INCLUDE[vs_ordesigner_short](../includes/vs-ordesigner-short-md.md)] contient un type de données qui n’est pas pris en charge par le [!INCLUDE[vs_ordesigner_short](../includes/vs-ordesigner-short-md.md)] (par exemple, [Types CLR définis par l’utilisateur](https://msdn.microsoft.com/library/9f70e0b0-3a0d-4eb1-b914-07a5d0c167c2)).  
  
### <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Créez une vue basée sur la table souhaitée qui n'inclut pas le type de données non pris en charge.  
  
2. Faites glisser la vue à partir de **Explorateur de serveurs**/**Database Explorer** sur le concepteur.
