# <a name="saving-entity-data"></a>Enregistrement d'Entity Data
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Lorsque vous créez, mettez à jour ou supprimez des entités qui se connectent à une base de données via WCF Data Services, vous devez explicitement enregistrer les modifications apportées à la base de données. Les rubriques suivantes décrivent comment enregistrer des modifications apportées aux données d’entité dans les applications WCF Data Services.  
  
|Rubrique|Description|  
|-----------|-----------------|  
|[Guide pratique pour ajouter, modifier et supprimer des entités](http://msdn.microsoft.com/library/a00f8933-b232-4445-95ba-adc634f055d8)|Montre comment enregistrer les modifications dans les données d’entité via un service de données.|  
|[Guide pratique pour exécuter les requêtes de services de données asynchrones](http://msdn.microsoft.com/library/902a2dc1-d0e9-4b00-90a8-becc4cb1f6a7)|Montre comment enregistrer des modifications via un service de données de façon asynchrone.|
