# <a name="supplying-xml-code-comments"></a>Insertion de commentaires dans le code XML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous pouvez insérer des commentaires dans les fichiers de code source IntelliSense. Quand vous ajoutez le commutateur /doc à la ligne de commande du compilateur Visual Basic, C# ou C++, un fichier .xml est généré pour servir de base à la documentation IntelliSense.  
  
 Pour plus d’informations sur la documentation du code dans Visual Basic, consultez [Comment : Créer une Documentation XML](http://msdn.microsoft.com/library/27b5b06c-09b9-496a-8245-f9542d846230).  
  
 Pour plus d’informations sur la documentation du code C#, consultez [Commentaires sur la documentation XML](http://msdn.microsoft.com/library/803b7f7b-7428-4725-b5db-9a6cff273199).  
  
 Pour plus d’informations sur la documentation du code C++, consultez [Commentaires](http://msdn.microsoft.com/library/6fcb906c-c264-4083-84bc-373800b2e514) et [Documentation XML](http://msdn.microsoft.com/library/a1aec1c5-b2d1-4c74-83ae-1dbbbb76b506).
