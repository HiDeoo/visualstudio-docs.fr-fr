# <a name="adding-and-removing-project-items"></a>Ajout et suppression d'éléments de projet
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous pouvez ajouter ou supprimer des projets et des éléments de projet en cliquant avec le bouton droit sur le nœud de projet ou de solution dans l’**Explorateur de solutions**, puis en choisissant **Ajouter** ou **Supprimer**. La suppression d'un projet d'une solution, ou d'un élément d'un projet, n'entraîne pas sa suppression du contrôle de code source ou du disque, sauf si vous sélectionnez l'option de suppression définitive.
