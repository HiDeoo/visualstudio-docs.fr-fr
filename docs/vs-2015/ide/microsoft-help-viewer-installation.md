# <a name="microsoft-help-viewer-installation"></a>Installation de Microsoft Help Viewer 2.0
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [Microsoft aide visionneuse Installation](https://docs.microsoft.com/visualstudio/ide/microsoft-help-viewer-installation).  
  
Plusieurs produits affichent l’aide dans Microsoft Help Viewer. Les instructions d'installation varient en fonction du produit qui a installé la visionneuse d'aide.
