# <a name="debugging-windows-store-and-windows-universal-apps"></a>Débogage d'applications du Windows Store et d'applications universelles Windows
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [débogage Windows Store et Windows applications universelles](https://docs.microsoft.com/visualstudio/debugger/debugging-windows-store-and-windows-universal-apps).  
  
Le débogueur Visual Studio fournit la prise en charge complète pour déboguer les applications du Windows Store.  
  
 Pour plus d’informations sur le débogage des applications Windows universelles, consultez [déploiement et débogage des applications de plateforme universelle Windows](https://msdn.microsoft.com/library/windows/apps/mt613243.aspx).  
  
 Pour déboguer les applications Windows Phone, consultez [débogage des applications pour Windows Phone 8](https://msdn.microsoft.com/library/windows/apps/ff402572\(v=vs.105\).aspx).
