# <a name="help-menu-commands"></a>commandes du menu Aide
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Le menu **aide** fournit des commandes permettant d’ouvrir des fichiers d’aide.  
  
 **Index**  
 Ouvre l’écran contenu de l’aide de Spy + +.  
  
 **Utilisation de l’aide**  
 Ouvre un fichier d’aide qui décrit comment utiliser l’aide.  
  
 **À propos de Spy++**  
 Affiche la version du logiciel et les informations de copyright pour Spy + +.
