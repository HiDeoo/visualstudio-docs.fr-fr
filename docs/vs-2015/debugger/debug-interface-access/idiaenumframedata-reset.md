# <a name="idiaenumframedatareset"></a>IDiaEnumFrameData::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [IDiaEnumFrameData::Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumframedata-reset).  
  
Réinitialise une séquence d’énumération au début.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Valeur de retour  
 Retourne `S_OK`.
