# <a name="window-menu-commands"></a>commandes du menu Fenêtre
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Le menu **fenêtre** fournit des commandes permettant de gérer les fenêtres Spy + +.  
  
 **Répercuté**  
 Réorganise toutes les fenêtres dans un modèle en cascade.  
  
 **Mosaïque horizontale**  
 Réorganisez toutes les fenêtres dans un modèle de mosaïque horizontal.  
  
 **Mosaïque verticale**  
 Réorganise toutes les fenêtres dans un modèle de mosaïque vertical.  
  
 **Réorganiser les icônes**  
 Réorganise les icônes de vue, si les fenêtres ont été réduites.  
  
 **Fermer/Fermer tout**  
 Ferme une ou toutes les fenêtres dans Spy + +.  
  
 **Actualiser**  
 Actualise la vue actuellement sélectionnée.  
  
 **Window**  
 Autorise la sélection d’une fenêtre spécifique.
