# <a name="managed-vspackages"></a>VSPackages gérés
Les rubriques suivantes expliquent comment créer un VSPackage. Un VSPackage est un module logiciel qui étend l' [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] environnement de développement intégré (IDE) en fournissant des éléments d’interface utilisateur, des services, des projets, des éditeurs et des concepteurs. Pour plus d'informations, consultez [VSPackages](../extensibility/internals/vspackages.md).
