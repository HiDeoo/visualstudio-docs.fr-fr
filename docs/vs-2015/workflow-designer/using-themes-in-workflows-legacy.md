# <a name="using-themes-in-workflows-legacy"></a>Utilisation de thèmes dans les workflows (hérités)
Cette rubrique explique comment utiliser des thèmes dans [!INCLUDE[wf](../includes/wf-md.md)] lors de l'utilisation du [!INCLUDE[wfd1](../includes/wfd1-md.md)] hérité qui cible le [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] ou le [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)].

 Vous pouvez personnaliser l'apparence de vos workflows avec le concepteur hérité en utilisant des thèmes [!INCLUDE[wf2](../includes/wf2-md.md)]. Un thème définit les couleurs d'arrière-plan et de premier plan, les styles, les icônes et les autres éléments visuels d'un workflow. Vous pouvez enregistrer des thèmes en vue de les réutiliser pour d'autres workflows.

 Vous créez et modifiez des thèmes à l’aide de la [boîte de dialogue Configuration du thème (hérité)](../workflow-designer/theme-configuration-dialog-box-legacy.md). Pour ouvrir cette boîte de dialogue, sélectionnez **créer un nouveau thème** dans le menu **Workflow** , ou cliquez avec le bouton droit sur l’aire de conception de workflow et sélectionnez **créer un nouveau thème**.

 Vous pouvez sélectionner les thèmes personnalisés par défaut et enregistrés à l’aide de la [boîte de dialogue thèmes, concepteur de flux de travail, options (héritée)](../workflow-designer/themes-workflow-designer-options-dialog-box-legacy.md). Pour ouvrir cette boîte de dialogue, cliquez sur **Sélectionner un thème personnalisé** dans le menu **Workflow** (ou cliquez avec le bouton droit sur l’aire de conception de workflow et cliquez sur **Sélectionner un thème personnalisé**), puis sélectionnez l’option **Concepteur de flux de travail** . Sélectionnez le thème de votre choix et cliquez sur **OK**.
