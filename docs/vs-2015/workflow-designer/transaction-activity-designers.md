# <a name="transaction-activity-designers"></a>Concepteurs d’activités de transaction

Les rubriques contenues dans cette section et répertoriées dans le tableau suivant fournissent des conseils sur la façon d'utiliser les concepteurs d'activités et de modèles [!INCLUDE[wfd2](../includes/wfd2-md.md)].
