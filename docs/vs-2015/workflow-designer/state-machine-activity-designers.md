# <a name="state-machine-activity-designers"></a>Concepteurs d'activités d'ordinateur d'état
[!INCLUDE[wfd1](../includes/wfd1-md.md)] comprend un certain nombre de concepteurs d’activités que vous pouvez utiliser lors de la construction de vos flux de travail. Cette section contient les concepteurs d'activités fournis par le système utilisés pour créer et gérer des workflows de machine à états. Les rubriques suivantes décrivent ces concepteurs et fournissent des conseils sur la façon de les utiliser.

## <a name="in-this-section"></a>Dans cette section
 [StateMachine](../workflow-designer/statemachine-activity-designer.md) Contient une collection d’États et de modèles de flux de travail à l’aide du paradigme d’ordinateur d’État familier.

 [État](../workflow-designer/state-activity-designer.md) Représente un État dans lequel une machine à États peut se trouver.

 [FinalState](../workflow-designer/finalstate-activity-designer.md) Représente un État qui termine une instance de machine à États.

 [Transition](../workflow-designer/transition-activity-designer.md) Représente la transition entre deux États.
