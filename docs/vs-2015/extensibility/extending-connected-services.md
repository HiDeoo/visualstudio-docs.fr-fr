# <a name="extending-connected-services"></a>Extension de services connectés
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Le kit de développement logiciel (SDK) Visual Studio Services connectés permet aux fournisseurs de services de créer des extensions Visual Studio que les développeurs d’applications peuvent facilement ajouter à leur projet sans quitter l’IDE de Visual Studio.
