# <a name="enabling-the-azure-sdk"></a>Activation du Kit de développement logiciel (SDK) Azure
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Le kit de développement logiciel (SDK) Azure pour .NET vous aide à créer, configurer, générer, déboguer, exécuter et empaqueter des applications et services Web évolutifs sur Azure.  
  
 Vous pouvez télécharger le kit de développement logiciel (SDK) Azure à partir de la boîte de dialogue **nouveau projet** .  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Pour activer le kit de développement logiciel (SDK) Azure  
  
1. Dans le menu **fichier** , créez un nouveau projet. (Cliquez sur **nouveau** , puis sur **projet**.)  
  
2. Développez le nœud pour **Visual Basic** ou **C#** , puis cliquez sur **Cloud**.  
  
3. Dans le volet central, sélectionnez **obtenir le kit de développement logiciel (SDK) Azure pour .net**, puis cliquez sur **OK**.  
  
     Suivez les instructions du programme d'installation.  
  
    > [!NOTE]
    > Pour installer les outils Azure, vous devez avoir Internet Information Services 7,0 et les composants de développement d’application ASP.NET installés. Si ces composants ne sont pas installés, le programme d’installation d’Azure Tools vous invite à les installer.
