# <a name="source-control"></a>Contrôle de code source
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Cette section décrit les options pour l’implémentation de contrôle de code source comme une fonctionnalité intégrée de [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)], via un plug-in de contrôle de code source ou d’un VSPackage de contrôle de code source.  
  
## <a name="in-this-section"></a>Dans cette section  
 [Éléments fondamentaux](../../extensibility/internals/source-control-integration-essentials.md)  
 Présente les informations importantes dont vous devez disposer pour commencer à travailler avec le contrôle de code source.  
  
 [Vue d’ensemble](../../extensibility/internals/source-control-integration-overview.md)  
 Vue d’ensemble des deux options disponibles pour l’implémentation de contrôle de code source.  
  
 [Création d’un plug-in de contrôle de code source](../../extensibility/internals/creating-a-source-control-plug-in.md)  
 Explique comment créer un plug-in de contrôle de code source qui fournit les fonctionnalités de contrôle de code source via le [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] interface utilisateur du contrôle source (IU).  
  
 [Création d’un VSPackage de contrôle de code source](../../extensibility/internals/creating-a-source-control-vspackage.md)
