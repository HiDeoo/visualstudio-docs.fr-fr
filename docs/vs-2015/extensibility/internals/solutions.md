# <a name="solutions"></a>Solutions
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Une solution est un regroupement d’un ou plusieurs projets qui fonctionnent ensemble pour créer une application. Les informations de projet et l’état qui se rapportent à la solution sont stockées dans deux fichiers de solution, un fichier textuel solution (.sln) et un fichier solution binaire des options (.suo) utilisateur.
