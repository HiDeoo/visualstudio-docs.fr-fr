# <a name="extensibility-in-visual-studio"></a>Extensibilité dans Visual Studio
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio vous fournit de nombreuses options d’extensibilité. Vous pouvez créer vos propres kits de développement logiciel (SDK), utiliser les kits de développement logiciel (SDK) Windows, Windows Phone et Azure (qui sont installés dans le cadre de Visual Studio) et étendre Visual Studio lui-même.

## <a name="extend-visual-studio"></a>Étendre Visual Studio
 Vous pouvez utiliser le kit de développement logiciel (SDK) Visual Studio pour étendre pratiquement toutes les parties de Visual Studio : commandes, menus, fenêtres, éditeurs et projets. Pour en savoir plus, consultez [Visual Studio SDK](../extensibility/visual-studio-sdk.md).

## <a name="create-your-own-sdks"></a>Créer vos propres kits de développement logiciel
 Découvrez comment créer, empaqueter et déployer vos propres kits de développement logiciel (SDK) de plateforme et d’extension : [création d’un kit de développement logiciel](../extensibility/creating-a-software-development-kit.md)(SDK).

## <a name="roslyn-extensibility"></a>Extensibilité Roslyn
 Découvrez comment utiliser Roslyn pour l’extensibilité : [.NET Compiler Platform (« Roslyn ») extensibilité](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).

## <a name="azure-sdk"></a>Azure SDK
 Découvrez comment activer le kit de développement logiciel (SDK) Azure : [activation du kit de développement logiciel (SDK) Azure](../extensibility/enabling-the-azure-sdk.md).
