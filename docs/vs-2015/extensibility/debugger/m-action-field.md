# <a name="maction-field"></a>Champ m_action
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [champ m_action](https://docs.microsoft.com/visualstudio/extensibility/debugger/m-action-field).  
  
Délégué qui représente le code à exécuter dans le <xref:System.Threading.Tasks.Task> objet.  
  
 **Namespace :** <xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **Assembly :** mscorlib (dans mscorlib.dll)  
  
 Étant donné que vous ne pouvez pas accéder à ce membre interne du .NET Framework, la syntaxe suivante est fournie en commun Intermediate Language (CIL).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
.field assembly object m_action  
```  
  
## <a name="remarks"></a>Notes  
 Il s’agit du `action` paramètre dans le <xref:System.Threading.Tasks.Task.%23ctor%2A> constructeur.
