# <a name="namematch"></a>NAME_MATCH
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [NAME_MATCH](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/name-match).  
  
Sélectionne l’option de casse pour les noms correspondants.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp#  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>Membres  
 nmNone  
 Aucune option n'est spécifiée.  
  
 nmCaseSensitive  
 Indique que les noms à mettre en correspondance respectent la casse.  
  
 nmCaseInsensitive  
 Indique que les noms à mettre en correspondance ne respectent pas la casse.
