# <a name="process-debug-manager"></a>Gestionnaire de débogage de processus
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [Gestionnaire de débogage de processus](https://docs.microsoft.com/visualstudio/extensibility/debugger/process-debug-manager).  
  
Le Gestionnaire de débogage de processus (PDM) est un composant de Visual Studio qui gère les programmes et processus, ce qui les rend disponibles pour la session de débogage manager et les moteurs de débogage.  
  
 PDM gère tous les processus qui peuvent être débogués. Pour être débogué, un programme doit être enregistré avec le PDM. Cela permet au moment que le programme est lancé par un port ou un moteur de débogage.
