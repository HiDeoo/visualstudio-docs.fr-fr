# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28160 : annotation d’erreur  
  
 Cet avertissement est signalé lorsqu’une `__drv_error` annotation a été rencontrée. Cette annotation est utilisée pour signaler les pratiques de codage qui doivent être corrigées et peut être utilisée avec une `__drv_when` annotation pour indiquer des combinaisons de paramètres spécifiques.
