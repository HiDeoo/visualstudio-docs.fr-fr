# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28021 : le paramètre \<param > annoté avec \<Anno > doit être un pointeur  
  
 Cet avertissement est signalé lorsque l’objet qui est annoté n’est pas un type pointeur. Cette annotation ne peut pas être utilisée avec des `void` ou des types intégraux.
