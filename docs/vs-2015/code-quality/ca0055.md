# <a name="ca0055"></a>CA0055

Fichier n’a pas pu être charger :\<chemin d’accès >.

L’outil d’analyse de code n’a pas pu charger le fichier cible qui a été spécifié pour l’analyse. Cela se produit si le fichier est introuvable ou l’accès au fichier a été refusé. En règle générale, cela se produit si l’utilisateur a choisi de substituer la propriété de l’assembly d’entrée dans Visual Studio, ou a appelé fxcopcmd.exe manuellement.
