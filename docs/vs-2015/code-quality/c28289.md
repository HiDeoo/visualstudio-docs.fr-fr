# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28289 : pour la fonction : ReadableTo ou WritableTo n’avait pas une spécification de limite en tant que paramètre  
  
 L’outil d’analyse de Code signale cet avertissement lorsque l’annotation du paramètre de la fonction est miscoded comme indiqué.
