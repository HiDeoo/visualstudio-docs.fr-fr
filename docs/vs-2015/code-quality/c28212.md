# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28212 : expression de pointeur attendue pour l’annotation  
  
 Cet avertissement indique que le paramètre numéroté d’une annotation (et non la fonction en cours d’annotation) est supposé être un type pointeur ou tableau, mais qu’un autre type a été rencontré. L’annotation doit être corrigée.
