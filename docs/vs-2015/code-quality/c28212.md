# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28212 : expression de pointeur attendue pour annotation  
  
 Cet avertissement indique que le paramètre numéroté à une annotation (pas la fonction annotée) doit être un type pointeur ou tableau, mais un autre type a été rencontrée. L’annotation doit être corrigée.
