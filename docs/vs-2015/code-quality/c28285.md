# <a name="c28285"></a>C28285
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28285 : pour la fonction, erreur de syntaxe dans le paramètre  
  
 L’outil d’analyse du code signale cet avertissement lorsqu’une erreur probable est rencontrée dans le fichier de modèle. Quelques erreurs de fichier source peuvent également provoquer de telles erreurs.
