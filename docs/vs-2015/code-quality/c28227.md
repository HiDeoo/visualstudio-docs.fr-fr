# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Avertissement C28227 : L'annotation ne peut pas être également un PrimOp (voir la déclaration antérieure)  
  
 Cet avertissement indique que l’annotation nommée est déclarée comme une annotation ordinaire et également a été précédemment déclarée comme un PrimOp. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être vu en cours d’utilisation classique.
