# <a name="c28236"></a>C28236
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28236](https://docs.microsoft.com/visualstudio/code-quality/c28236).  
  
avertissement C28236 : l’annotation ne peut pas être utilisée dans une expression  
  
 Cet avertissement indique qu’une tentative a été effectuée pour utiliser une fonction déclarée comme étant une annotation dans le contexte d’une expression. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être vu en cours d’utilisation classique.
