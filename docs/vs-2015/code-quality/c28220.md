# <a name="c28220"></a>C28220
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28220](https://docs.microsoft.com/visualstudio/code-quality/c28220).  
  
avertissement C28220 : expression d’entier attendue pour l’annotation du paramètre dans l’annotation  
  
 Cet avertissement indique qu’un paramètre à une annotation doit être une expression d’entier, et un autre type a été rencontrée. Cela indique généralement une macro d’annotation de code incorrect.
