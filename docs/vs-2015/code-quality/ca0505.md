# <a name="ca0505"></a>CA0505

La propriété déconseillée sera ignorée, car la propriété de remplacement est définie.

Avertissement CA505 est généré lorsque des déconseillé spécifié et remplacement propriétés d’analyse de code contiennent des valeurs dans un projet ou fichier de configuration de build. Pour résoudre ce problème, supprimez la propriété déconseillée.
