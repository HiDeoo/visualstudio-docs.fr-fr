# <a name="ca0505"></a>CA0505

La propriété déconseillée est ignorée, car la propriété de remplacement est définie.

Un CA505 d’avertissement est généré lorsque les propriétés d’analyse du code dépréciées et qui se superposent, contiennent des valeurs dans un projet ou un fichier de configuration de Build. Pour résoudre ce problème, supprimez la propriété déconseillée.
