# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28230](https://docs.microsoft.com/visualstudio/code-quality/c28230).  
  
avertissement C28230 : le type de paramètre n’a aucun membre.  
  
 Cet avertissement indique qu’un argument à une annotation tente d’accéder à un `struct`, `class`, ou `union` et le membre nommé n’existe pas.
