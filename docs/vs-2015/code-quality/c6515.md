# <a name="c6515"></a>C6515
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Avertissement C6515 : annotation non valide : \<nom > propriété peut uniquement être employée dans des valeurs de type pointeur  
  
 Cet avertissement indique qu’une propriété pour une utilisation sur les pointeurs a été appliquée à un type non pointeur. Pour obtenir la liste des propriétés d’annotation, consultez [propriétés de l’Annotation](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Exemple  
 Le code suivant génère cet avertissement :  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 Pour corriger cet avertissement, utilisez le code suivant :  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
