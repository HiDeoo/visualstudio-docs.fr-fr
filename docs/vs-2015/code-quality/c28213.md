# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28213 : l' `_Use_decl_annotations_` annotation doit être utilisée pour référencer, sans modification, une déclaration antérieure. \<why>  
  
 Cet avertissement est signalé lorsqu’une déclaration antérieure est référencée sans l' `_Use_decl_annotations_` annotation requise.
