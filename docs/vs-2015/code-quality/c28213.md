# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28213 : Le `_Use_decl_annotations_` annotation doit être utilisée pour référencer, sans modification, une déclaration antérieure. \<why>  
  
 Cet avertissement est émis lorsqu’une déclaration antérieure est référencée sans requis `_Use_decl_annotations_` annotation.
