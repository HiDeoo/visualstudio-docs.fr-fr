# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28213 : l’annotation de `_Use_decl_annotations_` doit être utilisée pour référencer, sans modification, une déclaration antérieure. \<pourquoi >  
  
 Cet avertissement est signalé lorsqu’une déclaration antérieure est référencée sans l’annotation `_Use_decl_annotations_` requise.
