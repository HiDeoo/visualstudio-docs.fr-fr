# <a name="c6995"></a>C6995
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C6995 : Impossible d’enregistrer le fichier journal XML  
  
 Cet avertissement indique que l’outil d’analyse du Code ne peut pas créer le journal des défauts, qui est la sortie de l’analyse du code.  
  
 Cette erreur peut indiquer une erreur disque ou indiquer que vous n’êtes pas autorisé à créer un fichier dans le répertoire spécifié.
