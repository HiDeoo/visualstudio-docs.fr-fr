# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28214](https://docs.microsoft.com/visualstudio/code-quality/c28214).  
  
avertissement C28214 : noms de paramètre d’attribut doivent être p1... p9  
  
 Cet avertissement indique que lorsque vous construisez une déclaration d’annotation, les noms de paramètres sont limités à p1... p9. Cet avertissement ne doit jamais se produire dans une utilisation normale.
