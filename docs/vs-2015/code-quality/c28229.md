# <a name="c28229"></a>C28229
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28229 : l’annotation ne prend pas en charge les paramètres  
  
 Cet avertissement indique qu’une annotation a été utilisée avec un paramètre lorsque l’annotation est déclarée sans paramètres. Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; Cet avertissement n’est pas censé être affiché dans un usage classique.
