# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28219 : membre d’énumération attendu pour l’annotation, le paramètre dans l’annotation  
  
 Un paramètre à une annotation est censé être un membre nommé `enum` type et certains autres symboles a été rencontrée ; utilisez un membre qui `enum` type. Cela indique généralement une macro d’annotation de code incorrect.
