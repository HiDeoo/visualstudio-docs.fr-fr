# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28280](https://docs.microsoft.com/visualstudio/code-quality/c28280).  
  
avertissement C28280 : pour le symbole, un 'end' a été trouvé sans mise en correspondance un 'begin'  
  
 Le langage d’annotation prend en charge un begin et end (`{` et `}` en C) construction et l’association est devenu déséquilibrée. Cette situation peut être évitée si les macros sont utilisées.
