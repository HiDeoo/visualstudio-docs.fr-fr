# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28223](https://docs.microsoft.com/visualstudio/code-quality/c28223).  
  
Avertissement C28223 : N’a pas trouvé de jeton/identificateur attendu pour l’annotation, paramètre  
  
 Cet avertissement indique qu’un paramètre à une annotation est censé être un identificateur, et certains autres symboles a été rencontrée. Cela indique généralement une macro d’annotation de code incorrect. L’utilisation de mots clés C ou C++ dans cette position provoquera cette erreur.
