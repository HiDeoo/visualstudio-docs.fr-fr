Vous trouverez la dernière version de cette rubrique dans [C1251](https://docs.microsoft.com/visualstudio/code-quality/c1251).  
  
avertissement C1251 : Impossible de charger des modèles.  
  
 L’outil d’analyse de Code signale cet avertissement lorsqu’il existe une erreur interne dans le fichier de modèle, et non dans le code en cours d’analyse.
