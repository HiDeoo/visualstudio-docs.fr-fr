# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement 28222 : \_Oui\_, \_non\_, ou \_peut-être\_ attendu pour annotation  
  
 Cet avertissement indique qu’un paramètre à une annotation doit être l’un des symboles `_Yes_`, `_No_`, ou `_Maybe_`, et certains autres symboles a été rencontrée. Cela indique généralement une macro d’annotation de code incorrect.
