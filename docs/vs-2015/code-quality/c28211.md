# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28211 : nom de contexte statique attendu pour SAL_context  
  
 Cet avertissement indique que l’opérande de l' `_Static_context_` annotation doit être le nom d’un contexte défini par l’outil. Cet avertissement ne doit jamais se produire normalement.
