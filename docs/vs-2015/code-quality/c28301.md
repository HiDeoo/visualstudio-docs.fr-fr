# <a name="c28301"></a>C28301
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28301 : aucune annotation pour la première déclaration de \<fonction >.\<Note1 > consultez \<nom de fichier > (\<ligne >). \<Note2 >  
  
 Cet avertissement est signalé lorsque les annotations sont introuvables dans la première déclaration d’une fonction donnée.
