# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28216 : l' \_ \_ annotation Check_return s’applique uniquement aux publications pour le paramètre de fonction spécifique.  
  
 L' `_Check_return_` annotation a été appliquée dans un contexte autre que `__post` ; elle peut nécessiter un `__post` `__drv_out` modificateur (ou), ou elle peut être placée de manière incorrecte.
