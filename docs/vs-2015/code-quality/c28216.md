# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28216](https://docs.microsoft.com/visualstudio/code-quality/c28216).  
  
avertissement C28216 : le _Check_return\_ annotation s’applique uniquement aux post-conditions pour le paramètre de fonction spécifique.  
  
 Le `_Check_return_` annotation a été appliquée dans un contexte autre que `__post`; elle peut avoir besoin une `__post` (ou `__drv_out`) modificateur, ou il peut être placés de manière incorrecte.
