# <a name="ca0064"></a>CA0064

Aucune analyse n’a été effectuée, car l’ensemble de règles spécifié ne contenait aucune règle FxCop.

Cet avertissement peut se produire dans l’une des situations suivantes :

- Si vous rencontrez cet avertissement conjointement avec des avertissements CA0063, un problème s’est produit lors du chargement de votre fichier RuleSet. Pour plus d’informations, consultez la documentation [CA0063](ca0063.md) .

- Si vous rencontrez cette erreur avec des avertissements CA0062, il est très probable que l’analyse du code n’a pas pu trouver l’assembly ou les assemblys qui contiennent les règles spécifiées par le RuleSet. Pour plus d’informations, consultez la documentation [CA0062](ca0062.md) .

- Dans le cas contraire, cet avertissement se produit lorsque votre ensemble de règles est vide ou que toutes les règles activées dans un ensemble de règles enfant ont été désactivées. Utilisez l’éditeur d' [ensembles de règles](../code-quality/working-in-the-code-analysis-rule-set-editor.md) de Visual Studio pour activer des règles dans votre ensemble de règles.
