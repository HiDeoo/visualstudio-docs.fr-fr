# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avertissement C28283 : pour le symbole, la spécification de taille indiquée n’est pas encore possible  
  
 L’avertissement indique que l’annotation sentinel « taille spécification » a reçu une valeur différente de zéro. Essentiellement, l’appelant a tenté de dire que la chaîne se termine par un caractère autre que zéro binaire.
