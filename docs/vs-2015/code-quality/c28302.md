# <a name="c28302"></a>C28302
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28302 : pour C++ le paramètre de référence < parameter_name >, un opérateur de `_Deref_` supplémentaire a été trouvé sur \<> d’annotation.  
  
 Cet avertissement est signalé lorsqu’un niveau supplémentaire de `_Deref_` est utilisé sur un paramètre.  
  
 SAL2 ne nécessite pas l’utilisation d’un niveau supplémentaire de `_Deref_` lors du traitement des paramètres de référence. Cette annotation particulière n’est pas ambiguë et est interprétée correctement, mais doit être corrigée.  
  
 Il est souvent possible de corriger ce risque en supprimant simplement l’ancienne `__deref` annotation et en utilisant la syntaxe SAL2. Parfois, il peut être nécessaire d’utiliser `_At_` pour référencer l’objet spécifique à annoter.
