# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28279](https://docs.microsoft.com/visualstudio/code-quality/c28279).  
  
avertissement C28279 : pour le symbole, un 'begin' a été trouvé sans 'end' correspondant  
  
 Le langage d’annotation prend en charge un début et fin (`{` et `}` en C) construction et l’association est devenu déséquilibrée. Cette situation peut être évitée si les macros sont utilisées.
