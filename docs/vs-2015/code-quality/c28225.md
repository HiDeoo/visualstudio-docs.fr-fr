# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28225 : le nombre correct de paramètres requis dans l’annotation est introuvable  
  
 Cet avertissement indique que l’annotation nommée est utilisée avec un nombre incorrect de paramètres. Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; Cet avertissement n’est pas censé être affiché dans un usage classique.
