# <a name="c28275"></a>C28275
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Vous trouverez la dernière version de cette rubrique dans [C28275](https://docs.microsoft.com/visualstudio/code-quality/c28275).  
  
avertissement C28275 : le paramètre de _Macro_value\_ a la valeur null  
  
 Cet avertissement indique qu’il existe une erreur interne dans le fichier de modèle, et non dans le code en cours d’analyse. Le *macroValue* fonction a été appelée sans aucun paramètre.
