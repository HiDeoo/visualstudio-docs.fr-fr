# <a name="c28275"></a>C28275
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

AVERTISSEMENT C28275 : le paramètre à \_Macro_value\_ a la valeur null  
  
 Cet avertissement indique qu’il y a une erreur interne dans le fichier de modèle, et non dans le code en cours d’analyse. La fonction *macroValue* a été appelée sans paramètre.
