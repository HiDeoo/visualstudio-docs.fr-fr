# <a name="how-to-view-managed-code-defects"></a>Procédure : afficher les défauts du code managé
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Dans l’analyse du code, les erreurs, les avertissements et les messages sont signalés dans la fenêtre analyse du code. La fenêtre analyse du code s’affiche automatiquement lorsque vous exécutez l’analyse du code sur un projet ou une solution.

 Dans la fenêtre analyse du code, vous pouvez :

- **Choisissez le titre de l’avertissement.**

     Des informations supplémentaires sur l’avertissement s’affichent dans la fenêtre analyse du code. La ligne du fichier source qui a déclenché l’avertissement est affichée et mise en surbrillance dans l’éditeur de code source.

- **Choisissez l’ID d’avertissement.** Les ID d’avertissement sont préfixés par l' **autorité de certification** et incluent un nombre à quatre chiffres. Par exemple, ca1234.

     La rubrique d’aide de la bibliothèque MSDN relative à l’avertissement s’affiche. La rubrique d’aide contient des informations détaillées sur la cause de l’avertissement et contient des exemples qui montrent comment le problème d’avertissement se produit et est résolu.
