# <a name="what39s-new-in-the-xslt-debugger"></a>Ce que&#39;s dans le débogueur XSLT
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Les fonctionnalités suivantes sont nouvelles dans Visual Studio 2010.  
  
## <a name="xslt-profiler"></a>Profileur XSLT  
 Le Générateur de profils XSLT est un outil qui permet aux développeurs de mesurer, d'évaluer et de cibler les problèmes liés aux performances dans le code XSLT en créant des rapports de performances XSLT détaillés. Pour plus d’informations, consultez [XSLT Profiler](../xml-tools/xslt-profiler.md).
