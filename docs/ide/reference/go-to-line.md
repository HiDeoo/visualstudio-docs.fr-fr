# <a name="go-to-line-dialog-box"></a>Atteindre la ligne (boîte de dialogue)

La boîte de dialogue **Atteindre la ligne** vous permet d’atteindre une ligne particulière dans le document actif. Pour accéder à cette boîte de dialogue, ouvrez un document à modifier, puis sélectionnez **Édition** > **Atteindre** > **Atteindre la ligne**, ou appuyez sur **Ctrl**+**G**.

![Boîte de dialogue Atteindre la ligne de Visual Studio](media/go-to-line-dialog-box.png)

## <a name="line-number-1---n"></a>Numéro de ligne (1 - \<n>)

Dans la zone **Numéro de ligne (1 - \<n>)**, entrez le numéro de la ligne à atteindre dans le document actif. Le numéro entré doit être compris entre 1 et le dernier numéro de ligne dans le document actif.
