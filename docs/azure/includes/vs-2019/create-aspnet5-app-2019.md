1. Dans la fenêtre de démarrage Visual Studio, sélectionnez **Créer un nouveau projet.**
1. Sélectionnez **ASP.NET application Web de base,** puis sélectionnez **Next**.
1. Entrez un nom pour votre nouvelle application (ou utilisez le nom par défaut), puis sélectionnez **Créer**.
1. Sélectionnez **application Web**.
1. Choisissez si vous voulez un support SSL en sélectionnant ou en dégageant la **zone de cocher de HTTPS.**
1. Sélectionnez la case à cocher **Enable Docker Support.**
1. Sélectionnez le type de conteneur que vous voulez (Windows ou Linux), puis sélectionnez **Créer**.
