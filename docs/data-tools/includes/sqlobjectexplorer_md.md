> [!NOTE]
>  Dans [!INCLUDE[vs_dev12](../../extensibility/includes/vs_dev12_md.md)], vous pouvez vous connecter à une base de données SQL Server à partir de le **Explorateur d’objets SQL Server** ou **des connexions de données** nœud **Explorateur de serveurs**. **Explorateur d’objets SQL Server** fournit une vue qui est similaire à SQL Server Management Studio. Pour ouvrir cet Explorateur dans la barre de menus, choisissez **vue**, **Explorateur d’objets SQL Server**. Pour plus d’informations, consultez [développement de base de données connectée](http://go.microsoft.com/fwlink/?LinkId=233646).
