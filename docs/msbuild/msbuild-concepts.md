# <a name="msbuild-concepts"></a>Concepts MSBuild

MSBuild fournit un schéma XML de base que vous pouvez utiliser pour contrôler la façon dont la plate-forme de construction construit un logiciel. Pour spécifier les composants de la génération et comment ils doivent être générés, utilisez ces quatre parties de MSBuild : propriétés, éléments, tâches et cibles.
