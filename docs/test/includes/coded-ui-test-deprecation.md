> [!NOTE]
> Les tests codés de l’interface utilisateur qui permettent d’effectuer des tests fonctionnels automatisés et pilotés par l’interface utilisateur sont dépréciés. Visual Studio 2019 est la dernière version dans laquelle les tests codés de l’interface utilisateur sont disponibles. Nous vous recommandons d’utiliser [Selenium](https://docs.seleniumhq.org/) pour tester les applications web et [Appium avec WinAppDriver](https://github.com/Microsoft/WinAppDriver) pour tester les applications de bureau et UWP.
