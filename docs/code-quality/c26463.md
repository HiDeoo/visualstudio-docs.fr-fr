# <a name="c26463-use_const_for_elements"></a>C26463 USE_CONST_FOR_ELEMENTS

Consultez [C++ Core instructions con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
