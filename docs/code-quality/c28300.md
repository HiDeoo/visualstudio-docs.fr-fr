# <a name="c28300"></a>C28300
avertissement C28300 : < nom_paramètre > : Opérandes d’expression de types incompatibles pour l’opérateur < nom_opérateur >

 Cet avertissement est émis lorsque les types d’opérandes dans un paramètre ne sont pas compatibles avec l’opérateur.
