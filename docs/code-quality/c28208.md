# <a name="c28208"></a>C28208
AVERTISSEMENT C28208 : La fonction \<function > a été précédemment définie avec une liste de paramètres différente à \<File > (\<Line >). Certains outils d’analyse génèrent des résultats incorrects

 Cet avertissement est signalé lorsque la définition connue d’une fonction ne correspond pas à une autre occurrence.
