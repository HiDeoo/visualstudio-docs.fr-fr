# <a name="c1251"></a>C1251
AVERTISSEMENT C1251 : Impossible de charger les modèles.

 L’outil d’analyse du code signale cet avertissement lorsqu’il y a une erreur interne dans le fichier de modèle, et non dans le code en cours d’analyse.
