# <a name="c28196"></a>C28196
AVERTISSEMENT C28196 : l’exigence n’est pas satisfaite. (L’expression n’est pas évaluée à true.)

 Cet avertissement indique que le prototype de fonction pour la fonction en cours d’analyse a une `__notnull`, `__null` ou `__drv_valueIs` sur un paramètre `_Out_` ou la valeur de retour, mais la valeur retournée est incohérente avec cette annotation.
