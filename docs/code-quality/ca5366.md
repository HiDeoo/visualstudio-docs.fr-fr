# <a name="ca5366-use-xmlreader-for-dataset-read-xml"></a>CA5366 : utiliser XmlReader pour lire le jeu de données XML

|||
|-|-|
|CheckId|CA5366|
|Category|Microsoft.Security|
|Modification avec rupture|Sans rupture|

## <a name="cause"></a>Cause

Une définition de type de document (DTD) définit la structure et les éléments et attributs légaux d’un document XML. Le fait de faire référence à une DTD à partir d’une ressource externe peut entraîner des attaques par déni de service (DoS) potentielles. La plupart des lecteurs ne peuvent pas désactiver le traitement DTD et limiter le chargement des références externes à l’exception de <xref:System.Xml.XmlReader?displayProperty=nameWithType> . L’utilisation de ces autres lecteurs pour charger du code XML à l’aide de l’une des méthodes suivantes déclenche cette règle :
- <xref:System.Data.DataSet.ReadXml%2A>
- <xref:System.Data.DataSet.ReadXmlSchema%2A>
- <xref:System.Data.DataSet.ReadXmlSerializable%2A>

## <a name="rule-description"></a>Description de la règle

L’utilisation <xref:System.Data.DataSet?displayProperty=nameWithType> d’un pour lire des données XML avec des données non fiables peut charger des références externes dangereuses, qui doivent être limitées à l’aide d’un <xref:System.Xml.XmlReader> avec un résolveur sécurisé ou lorsque le traitement DTD est désactivé.

## <a name="how-to-fix-violations"></a>Comment corriger les violations

Utilisez <xref:System.Xml.XmlReader> ou ses classes dérivées pour lire XML.

## <a name="when-to-suppress-warnings"></a>Quand supprimer les avertissements

Supprimez un avertissement de cette règle lors de la gestion d’une source de données approuvée.

## <a name="pseudo-code-examples"></a>Exemples de pseudo-code

### <a name="violation"></a>Violation

```csharp
using System.Data;
using System.IO;

public class ExampleClass
{
    public void ExampleMethod()
    {
        new DataSet().ReadXml(new FileStream("xmlFilename", FileMode.Open));
    }
}
```

### <a name="solution"></a>Solution

```csharp
using System.Data;
using System.IO;
using System.Xml;

public class ExampleClass
{
    public void ExampleMethod()
    {
        new DataSet().ReadXml(new XmlTextReader(new FileStream("xmlFilename", FileMode.Open)));
    }
}
```
