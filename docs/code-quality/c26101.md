# <a name="c26101"></a>C26101
AVERTISSEMENT C26101 : Impossible d’utiliser correctement l’opération d’interverrouillage pour la variable \<var >.

 Les API Windows offrent une variété d’opérations verrouillées. L’annotation `_Interlocked_` spécifie qu’une variable doit être accessible uniquement par le biais d’une opération verrouillée. L’avertissement C26101 est émis lorsqu’un accès n’est pas cohérent avec l’annotation `_Interlocked_`.

## <a name="example"></a>Exemple
 L’exemple suivant génère l’avertissement C26101 en raison d’une violation du contrat `_Interlocked_`.

```

CRITICAL_SECTION cs;
typedef struct _DATA
{
    _Interlocked_ LONG data;
} DATA;

void Safe(DATA* p)
{
    InterlockedIncrement(&p->data); // OK
}

void Unsafe(DATA* p)
{
    p->data += 1; // Warning C26101
    EnterCriticalSection(&cs);
    LeaveCriticalSection(&cs);
}
```
