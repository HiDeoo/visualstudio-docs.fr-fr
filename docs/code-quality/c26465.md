# <a name="c26465-no_const_cast_unnecessary"></a>C26465 NO_CONST_CAST_UNNECESSARY

N’utilisez pas `const_cast` pour effectuer un cast `const`. `const_cast` n’est pas obligatoire. constance ou la volatilité n'est pas supprimée par cette conversion. Consultez [Type.3 des recommandations C++ Core](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-constcast).
