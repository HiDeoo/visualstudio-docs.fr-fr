# <a name="c28214"></a>C28214
avertissement C28214 : Les noms des paramètres d'attribut doivent être p1...p9

 Cet avertissement indique que lorsque vous construisez une déclaration d’annotation, les noms de paramètres sont limités à p1... p9. Cet avertissement ne doit jamais se produire dans une utilisation normale.
