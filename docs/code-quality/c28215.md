# <a name="c28215"></a>C28215
AVERTISSEMENT C28215 : Le typefix ne peut pas être appliqué à un paramètre qui contient déjà un typefix

 L’application d’une annotation `__typefix` à un paramètre qui a déjà cette annotation est une erreur. Les annotations `__typefix` sont utilisées uniquement dans quelques cas spéciaux et cet avertissement n’est pas censé être affiché en mode d’utilisation normale.
