# <a name="c28021"></a>C28021
AVERTISSEMENT C28021 : le paramètre \<param > qui est annoté avec \<anno > doit être un pointeur

 Cet avertissement est signalé lorsque l’objet qui est annoté n’est pas un type pointeur. Cette annotation ne peut pas être utilisée avec des types `void` ou intégraux.
