# <a name="overview-of-code-analysis-for-managed-code-in-visual-studio"></a>Vue d’ensemble de l’analyse du code pour le code managé dans Visual Studio

Visual Studio peut effectuer une analyse de code du code managé de deux manières : avec [analyseurs binaire](../code-quality/walkthrough-analyzing-managed-code-for-code-defects.md), également connu comme l’analyse statique FxCop d’assemblys managés et avec les plus modernes [analyseurs de Roslyn](../code-quality/roslyn-analyzers-overview.md). Les analyseurs de Roslyn, laquelle analyser votre code en direct en cours de frappe, remplacent l’analyse statique FxCop, qui analyse uniquement votre code après une génération.
