# <a name="c28022"></a>C28022
avertissement C28022 : Les classes de fonction \<liste_classes_1 > sur cette fonction ne correspondent pas les classes de fonction \<liste_classes_2 > sur le typedef utilisé pour la définir.

 Cet avertissement est émis lorsqu’il existe une erreur dans les annotations. Le typedef et ont tous deux la fonction elle-même `_Function_class_` annotations, mais ils ne correspondent pas. Si les deux sont utilisés, ils doivent correspondre.
