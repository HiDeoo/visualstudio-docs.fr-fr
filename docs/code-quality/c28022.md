# <a name="c28022"></a>C28022
AVERTISSEMENT C28022 : la ou les classes de fonction \<classlist1 > sur cette fonction ne correspondent pas à la classe de fonction \<classlist2 > sur le typedef utilisé pour la définir.

 Cet avertissement est signalé en cas d’erreur dans les annotations. Le typedef et la fonction possèdent eux-mêmes des annotations `_Function_class_`, mais ils ne correspondent pas. Si les deux sont utilisés, ils doivent correspondre.
