# <a name="c28222"></a>C28222
AVERTISSEMENT 28222 : \_Yes @ no__t-1, \_No @ no__t-3 ou \_Maybe @ no__t-5 attendu pour l’annotation

 Cet avertissement indique qu’un paramètre d’une annotation est supposé être l’un des symboles `_Yes_`, `_No_` ou `_Maybe_`, et qu’un autre symbole a été rencontré. Cela indique généralement une macro d’annotation mal codée.
