# <a name="ca0505"></a>CA0505

La propriété déconseillée sera ignorée, car la propriété de remplacement est définie.

Avertissement CA505 est généré lorsque les deux les propriétés d’analysis déconseillées et remplacement de code ont des valeurs dans un projet ou générer le fichier de configuration. Pour résoudre ce problème, supprimez la propriété déconseillée.
