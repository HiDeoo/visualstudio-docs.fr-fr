# <a name="c28230"></a>C28230
AVERTISSEMENT C28230 : Le type de paramètre ne contient pas de membre.

 Cet avertissement indique qu’un argument d’une annotation tente d’accéder à un `struct`, `class` ou `union` et que le membre nommé n’existe pas.
