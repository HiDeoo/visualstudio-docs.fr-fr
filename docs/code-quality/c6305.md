# <a name="c6305"></a>C6305
AVERTISSEMENT C6305 : incompatibilité potentielle entre les quantités sizeof et countof

 Cet avertissement indique qu’une variable contenant un résultat `sizeof` est ajoutée ou soustraite d’un pointeur ou d’une expression `countof`. Cela entraînera une mise à l’échelle inattendue dans l’arithmétique de pointeur.

## <a name="example"></a>Exemple
 Le code suivant génère cet avertissement :

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 Pour corriger cet avertissement, utilisez le code suivant :

```
void f(int *p)
{
  // code...
  p += 1;
}
```
