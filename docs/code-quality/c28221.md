# <a name="c28221"></a>C28221
AVERTISSEMENT C28221 : expression de chaîne attendue pour le paramètre dans l’annotation

 Cet avertissement indique qu’un paramètre d’une annotation est supposé être une chaîne et qu’un autre type a été rencontré. Cela indique généralement une macro d’annotation mal codée et n’est pas censée être affichée dans un usage normal.
