# <a name="c28137"></a>C28137
AVERTISSEMENT C28137 : L’argument de variable doit plutôt être une constante (littérale)

 Cet avertissement est émis quand une constante (littéral) requise fait défaut dans un appel de fonction. Consultez la documentation de la fonction.

## <a name="example"></a>Exemple
 Par exemple, la routine [ExAcquireResourceExclusiveLite](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-exacquireresourceexclusivelite) requiert une valeur true ou false pour le paramètre `Wait`. L'exemple de code suivant génère cet avertissement :

```
ExAcquireResourceExclusiveLite(Resource, Wait);
```

 L'exemple de code suivant évite cet avertissement :

```
ExAcquireResourceExclusiveLite(Resource, TRUE);
```
