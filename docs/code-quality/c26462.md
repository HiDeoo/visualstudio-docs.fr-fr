# <a name="c26462-useconstpointerforvariable"></a>C26462 USE_CONST_POINTER_FOR_VARIABLE
  La valeur indiquée par « %variable% » est assignée une seule fois, marquez-la comme étant un pointeur vers `const`. Consultez [C++ Core instructions con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
