# <a name="c1257"></a>C1257
AVERTISSEMENT 1257 : Échec de l’initialisation du plug-in.

 L’outil d’analyse du code signale cet avertissement lorsqu’il y a une erreur interne dans le plug-in, et non dans le code en cours d’analyse.
