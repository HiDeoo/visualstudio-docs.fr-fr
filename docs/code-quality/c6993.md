# <a name="c6993"></a>C6993
avertissement 6993 : Analyse du code ignore les constructions OpenMP ; analyse du code monothread

 Cet avertissement indique que l'analyseur de code a rencontré des pragmas OpenMP qu'il ne peut pas analyser.
