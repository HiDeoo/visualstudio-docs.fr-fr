# <a name="c6515"></a>C6515
Avertissement C6515 : annotation non valide : \<nom > propriété peut uniquement être employée dans des valeurs de type pointeur

 Cet avertissement indique qu’une propriété pour une utilisation sur les pointeurs a été appliquée à un type non pointeur. Pour obtenir la liste des propriétés d’annotation, consultez [à l’aide d’Annotations SAL pour réduire les défauts du code](using-sal-annotations-to-reduce-c-cpp-code-defects.md).

## <a name="example"></a>Exemple
 Le code suivant génère cet avertissement :

```cpp
#include <sal.h>

void f(_Readable_bytes_(c) char pc,  size_t c)
{
    // code ...
}
```

 Pour corriger cet avertissement, utilisez le code suivant :

```
#include <sal.h>

void f(_Readable_bytes_(c) char * pc,  size_t c)
{
    // code ...
}
```
