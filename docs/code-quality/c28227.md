# <a name="c28227"></a>C28227
AVERTISSEMENT C28227 : L'annotation ne peut pas être également un PrimOp (voir la déclaration antérieure)

 Cet avertissement indique que l’annotation nommée est déclarée comme une annotation ordinaire, et qu’elle a également été déclarée précédemment en tant que PrimOp. Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; Cet avertissement n’est pas censé être affiché dans un usage classique.
