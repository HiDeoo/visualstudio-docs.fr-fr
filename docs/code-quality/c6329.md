# <a name="c6329"></a>C6329
AVERTISSEMENT C6329 : la valeur de retour d’un appel à \<function > ne doit pas être vérifiée par rapport à \<number >

 Le programme compare un nombre à la valeur de retour d’un appel à `CreateFile`. Si `CreateFile` parvient, il retourne un handle ouvert à l’objet. En cas d’échec, elle retourne `INVALID_HANDLE_VALUE`.

## <a name="example"></a>Exemple
 Ce code peut provoquer l’avertissement suivant :

```cpp
if (CreateFile() == NULL)
    {
        return;
    }
```

## <a name="example"></a>Exemple
 Ce code corrige l’erreur :

```cpp
if (CreateFile() == INVALID_HANDLE_VALUE)
    {
        return;
    }
```
