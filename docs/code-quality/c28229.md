# <a name="c28229"></a>C28229
avertissement C28229 : L'annotation ne prend pas en charge les paramètres

 Cet avertissement indique qu’une annotation a été utilisée avec un paramètre lors de l’annotation est déclarée sans paramètres. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être vu en cours d’utilisation classique.
