# <a name="c28241"></a>C28241
avertissement C28241 : L'annotation pour la fonction sur le paramètre n'est pas reconnue

 Un nom d’annotation non reconnu a été utilisé. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être vu en cours d’utilisation classique.
