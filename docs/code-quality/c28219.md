# <a name="c28219"></a>C28219
AVERTISSEMENT C28219 : Membre de l'énumération attendu pour une annotation, le paramètre dans l'annotation

 Un paramètre d’une annotation est supposé être un membre du type `enum` nommé et un autre symbole a été rencontré ; Utilisez un membre de ce type `enum`. Cela indique généralement une macro d’annotation mal codée.
