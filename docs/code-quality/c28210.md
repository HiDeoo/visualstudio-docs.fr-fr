# <a name="c28210"></a>C28210
AVERTISSEMENT 28210 : les annotations pour le contexte `_On_failure_` ne doivent pas être dans un contexte prédéfini explicite

 Les annotations `_On_failure_` doivent être indiquées explicitement ou implicitement dans le contexte `__post`, autrement dit, à appliquer après le retour de la fonction.  Utilisez `_drv_out` pour vous en assurer.
