# <a name="c28225"></a>C28225
avertissement C28225 : Nombre correct de paramètres requis introuvable dans l'annotation

 Cet avertissement indique que l’annotation nommée est utilisée avec un nombre incorrect de paramètres. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être vu en cours d’utilisation classique.
