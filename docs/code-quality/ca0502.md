# <a name="ca0502"></a>CA0502
Langage non pris en charge

 Cette erreur se produit lorsque l’utilisateur tente d’analyser un projet ASP.NET qui contient une ou plusieurs pages ou classes écrites dans une langue autre que c# ou Visual Basic. Pour corriger cette erreur, supprimez le fichier à partir du projet, ou désactiver l’analyse du code.
