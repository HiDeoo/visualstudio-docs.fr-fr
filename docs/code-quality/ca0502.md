# <a name="ca0502"></a>CA0502
Langue non prise en charge

Cette erreur se produit lorsque l’utilisateur tente d’analyser un projet ASP.NET qui contient une ou plusieurs pages ou classes écrites dans une langue autre C# que ou Visual Basic. Pour corriger cette erreur, supprimez le fichier du projet ou désactivez l’analyse du code.
