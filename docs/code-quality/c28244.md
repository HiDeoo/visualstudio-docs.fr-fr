# <a name="c28244"></a>C28244
avertissement C28244 : L’annotation pour la fonction a une annotation de paramètre/externe non analysable

 Cela doit actuellement pas possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être vu en cours d’utilisation classique.
