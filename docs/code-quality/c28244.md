# <a name="c28244"></a>C28244
AVERTISSEMENT C28244 : L’annotation pour la fonction a un paramètre ou une annotation externe non analysable

 Cela ne doit pas être possible actuellement si les macros standard sont utilisées pour les annotations ; Cet avertissement n’est pas censé être affiché dans un usage classique.
