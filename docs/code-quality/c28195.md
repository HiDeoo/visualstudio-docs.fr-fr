# <a name="c28195"></a>C28195
AVERTISSEMENT C28195 : La fonction a été déclarée comme obtenant de la mémoire dans une variable et a été quittée sans cela

 Cet avertissement indique que le prototype de fonction pour la fonction en cours d’analyse a une annotation `__drv_acquiresMemory`. L’annotation `__drv_acquiresMemory` indique que la fonction acquiert la mémoire à l’emplacement de résultat désigné, mais dans au moins un chemin d’accès, la fonction n’a pas acquis la mémoire. Notez que l’outil d’analyse du code ne reconnaît pas l’implémentation réelle d’un allocateur de mémoire (impliquant l’arithmétique des adresses) et ne reconnaît pas que la mémoire est allouée (bien que de nombreux wrappers soient reconnus). Dans ce cas, l’outil d’analyse du code ne reconnaît pas que la mémoire a été allouée et émet cet avertissement. Pour supprimer le faux positif, utilisez un avertissement `#pragma` sur la ligne qui précède l’accolade ouvrante `{` du corps de la fonction
