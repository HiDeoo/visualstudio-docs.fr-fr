# <a name="c28194"></a>C28194
AVERTISSEMENT C28194 : La fonction a été déclarée comme alias de la valeur dans la variable et a été quittée sans cela

 Cet avertissement indique que le prototype de fonction pour la fonction en cours d’analyse a une annotation `__drv_isAliased`, ce qui indique qu’il affectera un *alias* à l’argument spécifié (autrement dit, assignez la valeur de manière à ce qu’elle survit au retour de la fonction). Toutefois, la fonction n’effectue pas un alias de l’argument sur le chemin d’accès indiqué par l’annotation. La plupart des fonctions qui aliasent une variable enregistrent sa valeur dans une structure de données globale.
