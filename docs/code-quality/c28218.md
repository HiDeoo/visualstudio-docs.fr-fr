# <a name="c28218"></a>C28218
avertissement C28218 : Pour le paramètre de fonction, paramètre de l’annotation ne correspond pas trouvé dans le fichier

 Les annotations sur la ligne actuelle et sur la ligne dans le message ne sont pas cohérents. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être affichés dans une utilisation normale.
