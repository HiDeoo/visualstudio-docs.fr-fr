# <a name="c26464-use_const_pointer_for_elements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  Les valeurs pointées par les éléments du tableau'% array% 'ne sont assignées qu’une seule fois, marquez les éléments comme pointeur vers `const`. Consultez [ C++ les instructions principales con. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
