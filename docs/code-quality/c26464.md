# <a name="c26464-useconstpointerforelements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  Les valeurs désignées par les éléments du tableau '% tableau %' sont assignées qu’une seule fois, marquez les éléments en tant que pointeur vers `const`. Consultez [C++ Core instructions con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
