# <a name="c28213"></a>C28213
AVERTISSEMENT C28213 : l’annotation de `_Use_decl_annotations_` doit être utilisée pour référencer, sans modification, une déclaration antérieure.

`_Use_decl_annotations_` indique au compilateur d’utiliser les annotations d’une déclaration antérieure de la fonction.  Si aucune déclaration antérieure n’est trouvée, ou si la déclaration actuelle apporte des modifications aux annotations par rapport à l’émission de cet avertissement.


## <a name="example"></a>Exemple

```cpp
// from example.h
void example_func(_Out_writes_(n) char* buffer, int n);

// from example.cpp
_Use_decl_annotations_ 
void example_func(_Out_writes_z_(n) char* buffer, int n)
{
  // ...
  buffer[n] = '\0';
}
```

L’annotation de paramètre `buffer` ne correspond pas entre les deux fichiers.  Cela peut être résolu en modifiant l’annotation afin qu’elle corresponde à tous les emplacements, ou en supprimant toutes les annotations à l’exception de `_Use_decl_annotations_` de la définition de la fonction.  Dans cet exemple `_Out_writes_z_` semble être correct, nous allons le déplacer vers la déclaration de fonction dans le fichier d’en-tête.

```cpp

// from example.h
void example_func(_Out_writes_z_(n) char* buffer, int n);

// from example.cpp
_Use_decl_annotations_ 
void example_func(char* buffer, int n)
{
  // ...
  buffer[n] = '\0';
}
```
