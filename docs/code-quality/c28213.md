# <a name="c28213"></a>C28213
AVERTISSEMENT C28213 : l’annotation `_Use_decl_annotations_` doit être utilisée pour référencer, sans modification, une déclaration antérieure. @no__t 0why >

 Cet avertissement est signalé lorsqu’une déclaration antérieure est référencée sans l’annotation `_Use_decl_annotations_` requise.
