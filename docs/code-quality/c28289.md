# <a name="c28289"></a>C28289
AVERTISSEMENT C28289 : pour la fonction : ReadableTo ou WritableTo n’a pas de spécification de limite en tant que paramètre

 L’outil d’analyse du code signale cet avertissement lorsque l’annotation de fonction/paramètre est incodée comme indiqué.
