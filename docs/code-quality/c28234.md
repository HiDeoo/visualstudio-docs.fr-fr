# <a name="c28234"></a>C28234
AVERTISSEMENT C28234 : l’expression \_At @ no__t-1 ne s’applique pas à la fonction active

 Cet avertissement indique que la valeur d’une expression `_At_` n’identifie pas un objet accessible.
