# <a name="c28309"></a>C28309
AVERTISSEMENT 28309 : < parameter_name >: Les opérandes d’annotation doivent être de type Integer/enum/pointer. Les opérandes void C++ et les opérateurs surchargés ne sont pas pris en charge. Les valeurs float sont approximatives comme des entiers. Types : \<typelist >.

 Vous avez essayé d’utiliser une fonction void ou une fonction dans une expression d’annotation, et l’analyse du code ne peut pas la gérer.  Cette erreur se produit généralement lorsqu’un `operator==` qui est implémenté en tant que fonction est utilisé, mais d’autres cas peuvent également se produire. Examinez les types dans @no__t > 0typelist pour obtenir des indices sur ce qui est incorrect.
