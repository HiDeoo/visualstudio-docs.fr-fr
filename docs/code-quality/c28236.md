# <a name="c28236"></a>C28236
AVERTISSEMENT C28236 : L'annotation ne peut pas être utilisée dans une expression

 Cet avertissement indique qu’une tentative d’utilisation d’une fonction déclarée comme une annotation dans un contexte d’expression a été effectuée. Cela ne doit pas être possible si les macros standard sont utilisées pour les annotations ; Cet avertissement n’est pas censé être affiché dans un usage classique.
