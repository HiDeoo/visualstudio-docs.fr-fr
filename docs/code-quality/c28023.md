# <a name="c28023"></a>C28023
AVERTISSEMENT C28023 : La fonction affectée ou passée doit avoir une annotation `_Function_class_` pour au moins une des classes dans : \<classlist >

 Cet avertissement est généralement signalé quand une seule classe de fonction est utilisée et qu’un rappel du type approprié n’est pas déclaré.

 Cet avertissement est émis lorsque la fonction à gauche de l’assignation (ou de l’assignation implicite, s’il s’agit d’un appel de fonction) est annotée pour indiquer qu’il s’agit d’un type de fonction propre au pilote qui utilise l’annotation `_Function_class_` ou un typedef qui contient une telle Commentaire. La fonction sur le côté droit de l’assignation n’a pas d’annotation `_Function_class_`. La fonction à droite doit être annotée pour être du même type que la fonction sur la gauche. Pour ce faire, il convient généralement d’ajouter la déclaration \<class1 > \<funcname1 > avant la première déclaration de \<funcname2 >.
