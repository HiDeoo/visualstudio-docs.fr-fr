# <a name="c28246"></a>C28246
AVERTISSEMENT C28246 : l’annotation pour la fonction' \<name > '-parameter' \<parameter > 'ne correspond pas au type du paramètre

 Un opérateur __deref a été appliqué à un type qui n’est pas un pointeur lors de la création d’une annotation.
