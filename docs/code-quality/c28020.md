# <a name="c28020"></a>C28020
AVERTISSEMENT C28020 : l’expression \<expr > n’est pas vraie au niveau de cet appel

 Cet avertissement est signalé lorsque le \_satisfait\_ expression répertoriée n’a pas la valeur true. Souvent, cela indique un paramètre incorrect.

 Si cela se produit sur une déclaration de fonction, les annotations indiquent une condition impossible.
