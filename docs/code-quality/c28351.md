# <a name="c28351"></a>C28351
AVERTISSEMENT C28351 : l’annotation \<annotation > décrit où une valeur dynamique (une variable) ne peut pas être utilisée dans la condition.

 Cet avertissement est signalé lorsqu’une annotation est appliquée là où le typeC++ /C est inspecté.

 L’expression dans le `_When_` doit correspondre à une constante. La `_When_` est ignorée.
