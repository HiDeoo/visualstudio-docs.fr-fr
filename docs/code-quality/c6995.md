# <a name="c6995"></a>C6995
AVERTISSEMENT C6995 : échec de l’enregistrement du fichier journal XML

 Cet avertissement indique que l’outil d’analyse du code ne peut pas créer le journal des défauts, qui est la sortie de l’analyse du code.

 Cette erreur peut indiquer une erreur de disque ou indiquer que vous n’avez pas l’autorisation de créer un fichier dans le répertoire spécifié.
