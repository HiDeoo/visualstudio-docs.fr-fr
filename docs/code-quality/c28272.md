# <a name="c28272"></a>C28272
AVERTISSEMENT C28272 : l’annotation pour la fonction, paramètre lors de l’examen est incohérent avec la déclaration de fonction

 Cet avertissement indique une erreur dans les annotations, et non dans le code en cours d’analyse. Les annotations apparaissant sur une définition de fonction ne sont pas cohérentes avec celles qui apparaissent dans une déclaration. Les deux annotations doivent être résolues pour correspondre.
