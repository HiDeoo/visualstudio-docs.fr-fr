# <a name="c1255"></a>C1255
avertissement C1255 : Les données PCH du plug-in ont une longueur incorrecte.

 L’outil d’analyse de Code signale cet avertissement lorsqu’il existe une erreur interne dans l’outil, et non dans le code en cours d’analyse.
