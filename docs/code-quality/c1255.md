# <a name="c1255"></a>C1255
AVERTISSEMENT C1255 : La longueur des données PCH du plug-in est incorrecte.

 L’outil d’analyse du code signale cet avertissement lorsqu’il y a une erreur interne dans l’outil, et non dans le code en cours d’analyse.
