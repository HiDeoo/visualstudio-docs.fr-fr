# <a name="c26495-memberuninit"></a>C26495 MEMBER_UNINIT

La variable « %variable% » n’est pas initialisée. Initialisez toujours une variable membre. Consultez [Type.6 des recommandations C++ Core](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type).
