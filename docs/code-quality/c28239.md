# <a name="c28239"></a>C28239
avertissement C28239 : value, stringValue ou longValue, et paramn=xxx sont définis en même temps dans l'annotation sur le paramètre. Utiliser uniquement paramn=xxx

 Une erreur interne s’est produite dans le fichier de modèle PREfast. Cet avertissement ne doit pas avoir lieu en cours d’utilisation classique.
