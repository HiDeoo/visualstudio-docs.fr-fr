# <a name="c28239"></a>C28239
AVERTISSEMENT C28239 : l’annotation sur le paramètre a à la fois value, stringValue ou longValue ; et Param = xxx. Utiliser uniquement paramn=xxx

 Une erreur interne s’est produite dans le fichier de modèle PREfast. Cet avertissement ne doit pas se produire dans un usage classique.
