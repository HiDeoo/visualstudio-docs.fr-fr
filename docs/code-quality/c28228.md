# <a name="c28228"></a>C28228
avertissement C28228 : Paramètre d'annotation : impossible d'utiliser ce type dans les annotations

 Cet avertissement indique qu’un paramètre est de type qui n’est pas pris en charge. Annotations peuvent uniquement utiliser un ensemble limité de types comme paramètres. Cela ne doit pas être possible si les macros standards sont utilisées pour les annotations ; Cet avertissement n’est pas prévu pour être vu en cours d’utilisation classique.
