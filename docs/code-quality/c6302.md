# <a name="c6302"></a>C6302
Avertissement C6302 : incompatibilité de chaîne de format : une chaîne de caractères est passée en tant que paramètre \<number > quand une chaîne de caractères larges est requise dans l’appel à \<function >

 Cet avertissement indique que la chaîne de format spécifie qu’une chaîne de caractères larges est requise. Toutefois, une chaîne de caractères est passée. Ce défaut est susceptible de provoquer un incident ou un endommagement d’un formulaire.

## <a name="example"></a>Exemple
 L’exemple de code suivant génère cet avertissement, car une chaîne de caractères est passée à la fonction `wprintf` :

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%s", buff);
}
```

 L’exemple de code suivant utilise `%hs` pour spécifier une chaîne de caractères codés sur un octet avec la fonction `wprintf` :

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%hs", buff);
}
```

 L’exemple de code suivant utilise la fonction de manipulation de chaînes sécurisée `wprintf_s` pour corriger cet avertissement :

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf_s(L"%hs", buff);
}
```
