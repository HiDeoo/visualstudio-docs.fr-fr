# <a name="c1254"></a>C1254
AVERTISSEMENT C1254 : incompatibilité de version de plug-in : la version ne correspond pas à la version du pilote PREfast

 L’outil d’analyse du code signale cet avertissement en cas d’erreur interne avec la version du plug-in, et non dans le code en cours d’analyse.
