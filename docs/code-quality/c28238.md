# <a name="c28238"></a>C28238
AVERTISSEMENT C28238 : Plusieurs value, stringValue et longValue sont définies dans l'annotation sur le paramètre. Utiliser paramn=xxx

 Une erreur interne s’est produite dans le fichier de modèle PREfast. Cet avertissement ne doit pas se produire dans un usage classique.
