# <a name="c28284"></a>C28284
avertissement C28284 : Pour le symbole, prédicats sont actuellement pas pris en charge pour les symboles de fonction non

 Cet avertissement indique qu’une annotation conditionnelle (prédicat, `__drv_when`) a été trouvé sur autre chose qu’une fonction.
