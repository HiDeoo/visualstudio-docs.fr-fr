# <a name="c28284"></a>C28284
AVERTISSEMENT C28284 : pour Symbol, les prédicats ne sont actuellement pas pris en charge pour les symboles qui ne sont pas des fonctions

 Cet avertissement indique qu’une annotation conditionnelle (prédicat, `__drv_when`) a été trouvée sur autre chose qu’une fonction.
