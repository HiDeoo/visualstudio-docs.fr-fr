# <a name="c28308"></a>C28308
AVERTISSEMENT 28308 : La position d’argument de la liste de formats spécifiée par l’annotation est incorrecte.

 La position de l’argument de liste de format doit être un nom de paramètre ou un offset d’entier dans la liste de paramètres, ou zéro.

 Le deuxième paramètre de `IsFormatString2` (`where`) peut se présenter sous l’une des deux formes suivantes :

- Nom de paramètre, qui est considéré comme le premier argument de la chaîne de format.

- Décalage (`n`) par rapport au paramètre format-String.

  Dans la deuxième forme, le premier paramètre de chaîne de format est le @no__t argument 1/-0-TH après la chaîne de format. Si `n` est égal à zéro, des points de suspension sont spécifiés comme paramètre. Si vous spécifiez un décalage de zéro sans spécifier les points de suspension comme premier paramètre de chaîne de format, une erreur se produira.
