# <a name="c28302"></a>C28302
AVERTISSEMENT C28302 : Pour C++ le paramètre de référence < parameter_name >, un opérateur supplémentaire `_Deref_` a été trouvé sur \<annotation >.

 Cet avertissement est signalé lorsqu’un niveau supplémentaire de `_Deref_` est utilisé sur un paramètre.

 SAL2 ne nécessite pas l’utilisation d’un niveau supplémentaire de `_Deref_` pour traiter les paramètres de référence. Cette annotation particulière n’est pas ambiguë et est interprétée correctement, mais doit être corrigée.

 Il est souvent possible de corriger ce risque en supprimant simplement l’ancienne annotation `__deref` et en utilisant la syntaxe SAL2. Parfois, il peut être nécessaire d’utiliser `_At_` pour faire référence à l’objet spécifique à annoter.
