# <a name="c28282"></a>C28282
AVERTISSEMENT C28282 : les chaînes de format doivent être dans des conditions préalables

 Cet avertissement indique qu’une annotation `__drv_formatString` est trouvée, qui ne se trouve pas dans une annotation `_Pre_` (`__drv_in`) (les paramètres de fonction sont par défaut `_Pre_`). Vérifiez si l’annotation utilisée dans un bloc explicite avec une annotation `_Post_` (`__drv_out`). Si c’est le cas, supprimez l’annotation de tout bloc englobant qui l’a placée dans un contexte `_Post_`.
