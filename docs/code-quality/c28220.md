# <a name="c28220"></a>C28220
AVERTISSEMENT C28220 : expression d’entier attendue pour l’annotation le paramètre dans l’annotation

 Cet avertissement indique qu’un paramètre d’une annotation est supposé être une expression entière et qu’un autre type a été rencontré. Cela indique généralement une macro d’annotation mal codée.
