# <a name="spy-views"></a>Vues Spy++
Spy ++ affiche les vues de l’arborescence de windows, les messages, les processus et les threads disponibles sur votre ordinateur.

## <a name="in-this-section"></a>Dans cette section
 [Affichage de Windows](../debugger/windows-view.md) affiche une arborescence des contrôles et fenêtres disponibles.

 [La vue messages](../debugger/messages-view.md) affiche le flux de message associé à une fenêtre, un processus ou un thread.

 [Vue processus](../debugger/processes-view.md) affiche une arborescence des processus actifs.

 [Vue threads](../debugger/threads-view.md) affiche une arborescence des threads actifs.
