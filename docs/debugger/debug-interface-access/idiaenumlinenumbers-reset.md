# <a name="idiaenumlinenumbersreset"></a>IDiaEnumLineNumbers::Reset
Réinitialise une séquence d'énumération.

## <a name="syntax"></a>Syntaxe

```C++
HRESULT Reset();
```

## <a name="return-value"></a>Valeur de retour
 Retourne `S_OK`.
