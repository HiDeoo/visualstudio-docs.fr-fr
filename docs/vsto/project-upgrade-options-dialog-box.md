# <a name="project-upgrade-options-dialog-box"></a>Mise à niveau, boîte de dialogue Options projets
  Utilisez la page **Mise à niveau du projet** pour spécifier si Visual Studio modifie automatiquement les projets de complément VSTO pour cibler la version d’Office installée sur l’ordinateur de développement.

 Pour accéder à cette page, dans le menu **Outils** , cliquez sur **Options**. Développez le dossier **Outils Office** , puis cliquez sur **Mise à niveau du projet**.

## <a name="uielement-list"></a>Liste UIElement
 **Toujours mettre à niveau vers la version installée d’Office** cette option est désactivée pour [!INCLUDE[vs_dev12](../vsto/includes/vs-dev12-md.md)] , car les projets Office dans [!INCLUDE[vs_dev12](../vsto/includes/vs-dev12-md.md)] peuvent cibler uniquement [!INCLUDE[office14_long](../vsto/includes/office14-long-md.md)] ou Microsoft [!INCLUDE[Office_15_short](../vsto/includes/office-15-short-md.md)] applications. Pour plus d’informations, consultez [mise à niveau et migrer des solutions Office](../vsto/upgrading-and-migrating-office-solutions.md).
