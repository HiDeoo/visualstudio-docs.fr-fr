# <a name="overview-of-visual-studio-tools-for-ai"></a>Aperçu des outils visualiser les studios pour l’IA

Visual Studio Tools for AI est un environnement de développement intégré (IDE) que vous pouvez utiliser pour construire, tester et déployer des solutions d’apprentissage profond. En savoir plus sur la façon d’utiliser les outils AI à partir des tutoriels et des échantillons suivants.

## <a name="install-the-extension"></a>Installer l’extension

Consultez la page [Installer Visual Studio Tools pour IA](installation.md) pour apprendre à télécharger et installer l’extension.

## <a name="5-minute-quickstart"></a>5 minutes de démarrage rapide 

Apprenez à exécuter la solution d’apprentissage profond avec [TensorFlow et Python](tensorflow-local.md)

## <a name="step-by-step-tutorial"></a>Tutoriel étape par étape

Apprenez à construire une solution d’apprentissage profond à partir du [modèle Train a TensorFlow dans le](tensorflow-vm.md) tutoriel cloud.

## <a name="samples"></a>Exemples

Parcourez les échantillons dans le [référentiel Échantillons](https://github.com/Microsoft/samples-for-ai) et la [galerie Machine Learning](https://gallery.cortanaintelligence.com/projects).
