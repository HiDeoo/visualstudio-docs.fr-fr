# <a name="accessibility"></a>Accessibilité

Il existe un certain nombre d’outils et de fonctionnalités d’accessibilité intégrés directement dans macOS qui vous aideront à utiliser avec Visual Studio pour Mac.  En plus de ceux-ci, Visual Studio pour Mac a également les caractéristiques suivantes pour le rendre plus accessible pour les personnes de capacités variables:

* Élargissement du texte de l’éditeur
* Élargissement du texte Pad
* Personnalisation de thème de couleur d’éditeur de code
* Personnalisation des raccourcis clavier
* Navigation au clavier

> [!NOTE]
> Ce guide fait référence à Visual Studio pour Mac. Pour Visual Studio sur Windows, voir [les fonctionnalités d’accessibilité de Visual Studio](/visualstudio/ide/reference/accessibility-features-of-visual-studio).
