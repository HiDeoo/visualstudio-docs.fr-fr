# <a name="code-magazine-included-in-visual-studio-subscriptions"></a>CODE Magazine inclus dans les abonnements Visual Studio

CODE Magazine est un magazine indépendant de développeurs de logiciels couvrant les technologies de développement sur toutes les plateformes.  Les abonnements sélectionnés visual Studio peuvent recevoir un abonnement d’un an aux éditions imprimées ou numériques du magazine.

## <a name="activation-steps"></a>Étapes d’activation
Le processus de mise en place de votre abonnement CODE Magazine est simple.  Procédez comme suit :

1. visite[https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
2. Localisez la tuile code Magazine dans la section perfectionnement professionnel ou éducation de votre portail, et cliquez sur **Get Code**.
   > [!div class="mx-imgBorder"]
   > ![CODE Magazine Tile](_img/vs-code-magazine/vs-code-magazine-tile.png "Tuile code Magazine")
3. Vous verrez un message indiquant que votre code a été récupéré.  Cliquez sur **Activer** pour continuer.  (Si vous n’êtes pas prêt à activer, c’est correct.  Vous pouvez toujours retourner à la tuile pour activer.  Le lien "Get Code" se lit comme "Active" après avoir récupéré votre code.)
   > [!div class="mx-imgBorder"]
   > ![Code CODE Magazine récupéré](_img/vs-code-magazine/vs-code-magazine-success.png "Code récupéré avec succès")
4. Vous serez redirigé vers la page d’atterrissage du magazine CODE. Sélectionnez votre pays et choisissez comment vous souhaitez recevoir le magazine. 
   > [!NOTE]
   > Les membres Visual Studio Dev Essentials n’ont accès qu’à l’abonnement numérique. 
5. Sur la base de vos sélections, vous verrez une note indiquant le prix de votre abonnement ainsi que le montant du coupon qui a été appliqué.
   > [!NOTE]
   > Pour les abonnements imprimés :
   > - Les abonnés en dehors des États-Unis peuvent encourir des frais d’expédition. 
   > - Veuillez prévoir plusieurs semaines pour la livraison de votre première édition imprimée.
      > [!div class="mx-imgBorder"]
      > ![Détails de l’abonnement code Magazine](_img/vs-code-magazine/vs-code-magazine-details.png "Détails d’abonnement et prix")
6. Ensuite, vous devrez fournir vos informations d’abonné et une adresse de livraison.  (L’adresse de livraison est requise même pour les abonnements numériques, car elle prend en charge l’exécution, la sélection des offres et les processus d’accès numérique.)
7. Si vous habitez à l’extérieur des États-Unis et que vous avez choisi de recevoir un abonnement à un magazine imprimé, on peut vous demander de fournir une adresse de facturation et des informations de paiement pour couvrir les coûts d’expédition. 
8. Vous pouvez vous en aller à code HTML Newsletter en cliquant sur la case à cocher.
   > [!div class="mx-imgBorder"]
   > ![Informations sur les abonnés de CODE Magazine](_img/vs-code-magazine/vs-code-magazine-subscriber-info.png "Informations sur les abonnés et adresse de livraison")
9. Cliquez sur **Place Order** pour compléter votre abonnement.  
Une fois votre commande d’abonnement terminée, vous verrez une page de confirmation, y https://codemag.com/my/fulfillmentcompris un lien que vous pouvez utiliser pour voir les numéros que vous avez reçus : .

## <a name="frequently-asked-questions"></a>Forum aux questions
### <a name="q-if-the-subscription-is-free-why-am-i-being-asked-for-a-credit-card"></a>Q: Si l’abonnement est gratuit, pourquoi on me demande une carte de crédit?  
R : Il est prohibitif d’envoyer gratuitement des copies imprimées à l’échelle internationale.  Les abonnés qui choisissent l’édition imprimée du magazine CODE et résident en dehors des États-Unis seront facturés des frais d’expédition. Pour éviter les frais d’expédition, choisissez la version numérique du magazine. 

### <a name="q-why-do-i-need-to-provide-a-delivery-address-for-a-digital-subscription"></a>Q: Pourquoi dois-je fournir une adresse de livraison pour un abonnement numérique?
R : CODE Magazine collecte votre adresse, que l’abonnement soit pour un abonnement imprimé ou numérique pour prendre en charge l’exécution, la sélection des offres et les processus d’accès numérique.  Pour plus d’informations sur ce que CODE Magazine fait avec vos données, veuillez consulter leur [politique de confidentialité](https://www.codemag.com/Home/Privacy).
