# <a name="the-rstudio-desktop-benefit-included-in-visual-studio-enterprise-subscriptions"></a>L’avantage RStudio Desktop inclus dans les abonnements Visual Studio Enterprise

Prenez le contrôle de votre code R.  RStudio est un environnement de développement intégré (IDE) pour R. Il comporte une console, un éditeur avec mise en surbrillance de la syntaxe prenant en charge l’exécution de code directe, ainsi que des outils de gestion du traçage, de l’historique, du débogage et de l’espace de travail. Voir d’autres [fonctionnalités de RStudio](https://www.rstudio.com/products/rstudio/features/).

Disponible en éditions commerciales et open source, RStudio s’exécute sur l’ordinateur de bureau (Windows, Mac et Linux) ou dans un navigateur connecté à RStudio Server ou RStudio Server Pro (Debian/Ubuntu, RedHat/CentOS et SUSE Linux).  L’avantage RStudio Desktop inclus dans les abonnements Visual Studio Enterprise consiste en un an d’accès à l’édition commerciale.

## <a name="activation-steps"></a>Étapes d’activation
1. Pour activer votre abonnement, connectez-vous à [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs), puis cliquez sur le lien **Obtenir un code** dans la vignette de l’avantage RStudio qui se trouve dans la section Outils.
   > [!div class="mx-imgBorder"]
   > ![Vignette RStudio](_img/vs-rstudio/vs-rstudio-tile.png)

2. Vous recevez une notification confirmant l’envoi du code du coupon.  Cliquez sur **Activer**.  Si vous choisissez **Annuler**, vous pourrez toujours activer votre avantage RStudio plus tard.  Une fois le code obtenu, le lien sur la vignette devient **Activer** ; vous pourrez cliquer dessus plus tard pour poursuivre le processus d’activation.

3. Vous accédez ensuite automatiquement à la page RStudio correspondant à l’avantage inclus dans votre abonnement Visual Studio.

    > [!NOTE]
    > En haut de la page d’accueil se trouve une section intitulée Configurez votre abonnement, permettant d’acheter des licences supplémentaires pour l’édition commerciale de RStudio Desktop.  Votre abonnement comporte une licence.  Entrez une quantité dans le champ vide **uniquement** si vous souhaitez acheter une licence supplémentaire.

    > [!div class="mx-imgBorder"]
    > ![Page d’arrivée de RStudio](_img/vs-rstudio/vs-rstudio-landing-page.png)

4. Le code de votre avantage est renseigné automatiquement sur la page.  S’il ne s’applique pas automatiquement, cliquez sur le bouton **Appliquer le code**. La Synthèse de l’abonnement, en haut à droite de la page, reflète alors la remise et indique **Total pour aujourd’hui** 0,00 $.

5. Remplissez les sections « Informations client », « Informations d’expédition », « Informations de facturation », « Adresse de facturation » et « Informations complémentaires » de la page.

    > [!IMPORTANT]
    > En souscrivant l’avantage de RStudio Desktop, vous vous inscrivez à un abonnement annuel périodique.  Le prix d’achat de cet abonnement sera débité sur votre carte de crédit, sauf si vous annulez votre abonnement avant la date de renouvellement.

6. Cochez la case pour accepter les **Conditions générales**.

7. Cliquez sur **Passer commande**.

    Après avoir passé commande, vous recevez plusieurs e-mails importants, notamment les informations concernant la gestion de votre compte et l’accès au support.

8. Vous pouvez maintenant procéder à l’installation de RStudio.  Suivez pour cela les invites de l’Assistant.

    > [!div class="mx-imgBorder"]
    > ![Programme d’installation de RStudio](_img/vs-rstudio/vs-rstudio-installer.png)

    > [!NOTE]
    > Vous devez disposer d’une installation valide de R sur votre ordinateur pour pouvoir lancer RStudio.  Si R n’est pas installé, vous recevrez le message d’erreur suivant.  Cliquez sur **Oui** pour télécharger R à partir du site web officiel du R Project, ou téléchargez et installez R à partir d’une autre source avant d’utiliser RStudio.

   > [!div class="mx-imgBorder"]
   > ![Message d’erreur : R non installé](_img/vs-rstudio/vs-rstudio-not-installed.png)

9. Dans « Bienvenue dans l’Assistant Installation de RStudio », cliquez sur **Suivant**.

    > [!div class="mx-imgBorder"]
    > ![Assistant Installation de RStudio](_img/vs-rstudio/vs-rstudio-wizard.png)

10. Choisissez le dossier de destination ou acceptez la valeur par défaut proposée par l’Assistant, puis cliquez sur **Suivant**.
    > [!div class="mx-imgBorder"]
    > ![Dossier de destination de RStudio](_img/vs-rstudio/vs-rstudio-destination.png)

11. Choisissez le dossier du Menu Démarrer ou acceptez la valeur par défaut, puis cliquez sur **Installer**.
    > [!div class="mx-imgBorder"]
    > ![Dossier du menu Démarrer de RStudio](_img/vs-rstudio/vs-rstudio-start-folder.png)

12. Une fois l’installation terminée, cliquez sur **Terminer** pour quitter le programme d’installation.
    > [!div class="mx-imgBorder"]
    > ![Installation de RStudio terminée](_img/vs-rstudio/vs-rstudio-complete.png)

13. Recherchez RStudio dans le dossier du menu Démarrer spécifié, ou tapez « RStudio » dans la zone de recherche Windows pour localiser l’application et lancer RStudio.
    > [!div class="mx-imgBorder"]
    > ![Tableau de bord de RStudio](_img/vs-rstudio/vs-rstudio-dashboard.png)

## <a name="frequently-asked-questions"></a>Questions fréquentes (FAQ)

### <a name="q-i-dont-want-to-sign-up-for-a-recurring-subscription--can-i-opt-out"></a>Q : Je ne veux pas souscrire un abonnement périodique.  Puis-je refuser ?

R :  L’inscription à l’abonnement périodique se fait automatiquement, mais vous pouvez l’annuler à tout moment avant la date de renouvellement tout en bénéficiant gratuitement de la première année de RStudio incluse avec votre abonnement Visual Studio.

Après avoir effectué l’inscription à RStudio, vous recevrez plusieurs e-mails, dont un vous invitant à gérer votre abonnement à RStudio sur BillingPortal.com.  Acceptez l’invitation pour modifier vos abonnements et gérer les informations bancaires stockées et les détails de votre compte.

### <a name="q--if-i-dont-have-r-installed-where-can-i-get-it"></a>Q :  Où peut-on obtenir R s’il n’est pas installé ?

R :  Si vous recevez une erreur indiquant que R n’est pas installé, le message d’erreur contiendra le lien de « Comprehensive R Archive Network (CRAN) » à l’adresse https://cran.rstudio.com/.  Vous pouvez également rechercher d’autres emplacements de téléchargement sur Internet.
