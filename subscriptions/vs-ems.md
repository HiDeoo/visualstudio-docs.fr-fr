# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Avantage EMS (Enterprise Mobility + Security) dans les abonnements Visual Studio

Enterprise Mobility + Security, qui a été précédemment disponible indépendamment dans les abonnements Visual Studio sélectionnés, est désormais inclus dans l’abonnement Microsoft 365 Developer inclus dans les mêmes abonnements Visual Studio sélectionnés.  

Par conséquent, les Enterprise Mobility + Security ne seront plus proposées séparément.  

Pour activer votre abonnement Microsoft 365 Developer :
1. Connectez-vous au [portail des abonnements Visual Studio](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)
1. Recherchez la vignette Microsoft 365 Developer Subscription, puis cliquez sur **activer**.  
1. Suivez les invites pour terminer la création de votre compte.  

Pour plus d’informations sur l’activation de [Microsoft 365 abonnement développeur](vs-m365.md).
