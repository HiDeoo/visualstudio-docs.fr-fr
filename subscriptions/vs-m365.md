# <a name="microsoft-365-developer-subscription-in-visual-studio-subscriptions"></a>Abonnement microsoft 365 développeur dans les abonnements Visual Studio
À partir de septembre 2019, tous les **nouveaux** membres du programme de développement Office 365 (y compris les abonnés Visual Studio Professional et Enterprise) peuvent s’inscrire à un abonnement Microsoft 365 E5 (Windows non inclus). Cela remplace l’abonnement de développeur E3 Office 365 que nous avons mis à disposition à partir de mars 2018. Il comprend maintenant EMS, quelque chose que nos clients nous ont demandé sur User Voice.  Consultez le récent [developer Blog](https://developer.microsoft.com/graph/blogs/new-microsoft-365-e5-subscription-with-ems-now-available-for-developers/) pour plus d’informations.

Les abonnements standard sélectionnés Visual Studio Professional ou Enterprise auront désormais accès à l’avantage d’abonnement microsoft 365 tant que leurs abonnements seront actifs.

## <a name="what-is-different-about-e5-vs-e3"></a>Qu’y a-t-il de différent dans l’E5 vs E3 ?
L’abonnement Microsoft 365 E5 développeur offre une nouvelle valeur et un accès élargi à la charge de travail critique Microsoft 365 et les capacités. Les développeurs avec microsoft 365 E5 abonnements bac à sable peuvent obtenir tout ce qu’ils ont apprécié dans E3 plus peut profiter de:

- Analyse avancée avec Power BI
- Mobilité d’entreprise et sécurité (SME) pour la conformité et la protection de l’information
- Office 365 Protection avancée des menaces
- Azure Active Directory pour la construction de solutions avancées de gestion de l’identité et de l’accès

Pour voir ce qui est inclus dans l’abonnement Microsoft 365 E5 développeur, voir [Comparer Microsoft 365 plans d’entreprise](https://nam06.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.microsoft.com%2Fen-us%2Fmicrosoft-365%2Fcompare-all-microsoft-365-plans&data=04%7C01%7Cmikeald%40microsoft.com%7Cab7e48abb3a04d87278308d716e495b8%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C637003043338656615%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C-1&sdata=NO9irwnRgVZme0nuZj50X7mI0G3bEeUs2ZdwYuxEr%2B8%3D&reserved=0) et [Microsoft 365 Volume Licensing](https://nam06.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.microsoft.com%2Fen-us%2Flicensing%2Fproduct-licensing%2Fmicrosoft-365-enterprise&data=04%7C01%7Cmikeald%40microsoft.com%7Cab7e48abb3a04d87278308d716e495b8%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C637003043338666614%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C-1&sdata=WziZov9B702H%2BxTchPPAmJmIpAsc6zbetlWOtFpANSk%3D&reserved=0).

Pour des descriptions détaillées des deux versions, consultez notre [documentation mise à jour](/office/developer-program/office-365-developer-program-faq).

## <a name="activation-steps"></a>Étapes d’activation
1. Connectez-vous à [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).

0. Localiser la tuile d’avantage Microsoft 365 dans la catégorie "Tools" et cliquez sur **Activate**.

0. Sur l’écran de bienvenue, choisissez votre région/pays, entrez le nom de votre entreprise et cliquez sur la case à cocher pour accepter les conditions d’utilisation.  Ensuite, cliquez sur **Suivant**.

0. Sur l’écran préférences du programme, remplissez les différents domaines pour indiquer vos préférences et domaines d’intérêt. Lorsque vous avez terminé, cliquez sur **Activez l’abonnement**.

0. Dans le **dialogue d’abonnement Set up votre développeur,** choisissez votre pays, créez votre nom d’utilisateur et votre domaine, et définissez votre mot de passe.  Puis, cliquez sur **Continuer**.

0. Spécifiez votre nom de code d’un numéro de téléphone qui prend en charge les messages SMS, et cliquez sur **Envoyer le code**.  Vous recevrez un code via un message SMS. 

0. Entrez le code que vous avez reçu dans le message texte et cliquez sur **Configurez.**

0. Vous verrez la page Office Dev Center, avec vos informations d’abonnement affichées. C’est une bonne idée de profiter de cette occasion pour attribuer des abonnements à vos utilisateurs.  Cliquez **sur Aller à l’abonnement**.

0. Connectez-vous à votre compte. 

0. Annulez la boîte de dialogue d’introduction.

0. Cliquez **sur Got it** pour fermer le dialogue Office 365 applications. 

0. Cliquez sur le bouton **Admin** pour ouvrir le centre d’administration Microsoft 365. 

0. Sélectionnez **les utilisateurs** dans le volet nav gauche, puis choisissez **les utilisateurs actifs**.  

0. Une liste de vos utilisateurs actifs apparaîtra.  Si vous venez de configurer votre abonnement, vous serez le seul répertorié.  Cliquez sur **Add a user**.

0. On peut vous demander de personnaliser votre connexion et votre courriel.  Choisissez les options qui fonctionnent le mieux pour vous, et cliquez sur **Next**. 

0. Sur la page **Ajouter de nouveaux utilisateurs,** vous verrez une grille que vous pouvez utiliser pour ajouter des utilisateurs qui auront tous des licences qui leur seront attribuées.  Si vous avez besoin d’ajouter plus d’utilisateurs cliquez **sur - Ajouter un autre utilisateur**.

Après avoir ajouté des utilisateurs, vous aurez la possibilité de faire des choses comme créer des équipes et installer des applications.  Vous pouvez soit faire tout ou toutes ces choses maintenant, ou revenir au centre d’administration à tout moment pour les configurer.

## <a name="frequently-asked-questions"></a>Forum aux questions
### <a name="q-how-do-current-users-upgrade-to-e5"></a>Q: Comment les utilisateurs actuels se mettent-ils à l’E5 ?
R : À l’heure actuelle, seuls les nouveaux membres du Programme des développeurs Office 365 recevront automatiquement l’offre E5. Rassurez-vous, nous nous efforçons de permettre à tous les membres du programme de développement Office 365 de passer de l’E3 à l’E5. Restez à l’écoute pour plus d’informations dans les prochains mois.

### <a name="q-can-i-activate-a-microsoft-365-developer-subscription-using-an-existing-tenant-or-do-i-need-to-create-a-new-one"></a>Q: Puis-je activer un abonnement Microsoft 365 développeur en utilisant un locataire existant, ou dois-je créer un nouveau?
R : Pour activer l’abonnement Office Developer, un nouveau locataire est requis.  

### <a name="q-can-i-choose-e3-or-e5"></a>Q: Puis-je choisir l’E3 ou l’E5?
En septembre 2019, les nouveaux membres du Programme des développeurs ne se voient offrir que l’E5, qui est notre version phare d’Office 365.  Cette version d’Office 365 offre encore plus de valeur que l’E3, et peut normalement coûter beaucoup plus cher que notre abonnement E3 précédent.

Les membres actuels de l’E3, qui n’ont pas besoin des capacités supplémentaires d’E5, auront la possibilité de conserver leur abonnement E3 actuel et de continuer à le renouveler. Nous n’exigeons pas qu’un membre existant passe de l’E3 à l’E5 s’il ne veut pas migrer. Toutefois, chaque membre ne peut avoir qu’un seul abonnement.

Lorsque nous avons une option de migration pour les membres actuels du Programme des développeurs, nous vous fournirons plus d’informations sur la façon de choisir et sur la façon de migrer.

### <a name="q-how-do-i-know-which-version-i-have"></a>Q: Comment puis-je savoir quelle version j’ai?
Vous pouvez vérifier votre version dès maintenant dans la tuile d’abonnement située sur votre tableau de bord Office 365 Developer Program.

Recherchez l’indication E3 ou E5 dans votre tuile.

### <a name="q-if-microsoft-365-developer-subscription-doesnt-include-windows-how-do-i-get-windows"></a>Q : Si l’abonnement microsoft 365 ne comprend pas Windows, comment puis-je obtenir Windows ?
R : Les abonnements Visual Studio qui incluent l’avantage Microsoft 365 incluent également Windows dans le cadre des téléchargements d’abonnés.  Visitez [les téléchargements](https://my.visualstudio.com/downloads) pour télécharger Windows. (Si vous ne voyez pas les téléchargements activés dans votre abonnement, veuillez [contacter votre administrateur](contact-my-admin.md).)
